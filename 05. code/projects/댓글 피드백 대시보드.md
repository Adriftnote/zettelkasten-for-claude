# 댓글 피드백 대시보드

온드미디어 채널 댓글/피드백 데이터를 Metabase 대시보드로 시각화하고, Excel → DB 증분 동기화하는 프로젝트

## 개요

브랜드커뮤니케이션팀에서 관리하는 온드미디어 채널(네이버 블로그, 인스타그램, 유튜브 숏츠, 티스토리, 틱톡) 댓글을 수집·분류하여 Metabase 대시보드로 제공합니다. Excel 원본을 DB에 증분 동기화하는 GUI 도구도 포함합니다.

## 코드 구성

**모듈**
- sync-feedback-gui: Excel → NAS DB 증분 동기화 GUI (EXE 배포)

**함수 (동기화)**
- read-feedback-excel: Excel 피드백 데이터 파싱
- get-existing-records: NAS DB 기존 레코드 조회
- verify-existing: 기존 데이터 정합성 검증
- insert-feedback-records: 신규 레코드 증분 INSERT

**함수 (인프라)**
- nas-ssh-connect: paramiko SSH 연결 (비밀번호 내장)
- save-sync-log: 동기화 이력 DB 저장

## 인프라

- DB: NAS feedback.db (Metabase DB ID:3)
- 대시보드: Metabase 대시보드 ID:4
- EXE: `outputs/피드백_동기화.exe` (38MB)
