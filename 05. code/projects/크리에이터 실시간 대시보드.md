# 크리에이터 실시간 대시보드

YouTube, TikTok, Meta(Facebook+Instagram), Naver(TV+Blog) 4개 플랫폼의 실시간 조회수와 채널 성과를 모니터링하는 대시보드. Metabase 대체 목적으로 Go(Gin) + HTML + ECharts로 구축.

## 개요

Go Gin 서버가 HTML 템플릿을 렌더링하고, 프론트엔드가 JSON API를 1분 주기로 폴링하여 ECharts 차트와 테이블을 갱신한다. 현재 더미 데이터로 프론트엔드를 먼저 구현하는 단계이며, 이후 MariaDB 연동 예정. 10분 간격 시계열 데이터(144포인트 = 24시간)를 사용하며, 차트에 endLabel로 현재값을 항상 표시한다.

## 아키텍처

```
Browser ──1분 폴링──→ GET /api/* ──→ Gin Router ──→ handlers/api.go ──→ dummy/data.go
                                                                         (→ MariaDB 예정)
        ←── JSON ←──────────────────────────────────────────────────────────
        ←── HTML ←── GET / /youtube /tiktok /meta /naver ──→ handlers/pages.go ──→ templates/*.html
```

## 페이지 구성

| 경로 | 페이지 | 주요 표시 |
|------|--------|-----------|
| / | Overview | 4채널 오늘 누적 조회수 + 10분 미니차트 + 컨텐츠별 채널 조회수 테이블 |
| /youtube | YouTube Studio | 채널 정보, Views/WatchTime/Subs 차트, 비디오 테이블 |
| /tiktok | TikTok Creator | 유저 정보, Views&Reach/Engagement 차트, 포스트 테이블 |
| /meta | Meta Business | 페이지 정보, Ad Impressions 차트, 오가닉+광고 테이블 |
| /naver | Naver | TV/Blog 정보, 재생수/방문자 차트, 클립+블로그 테이블 |
