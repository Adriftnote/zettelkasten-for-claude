# 📊 Extraction Status

## 미추출 (Pending)

```dataview
TABLE WITHOUT ID
  file.link AS "Source",
  file.folder AS "Category"
FROM "03. sources"
WHERE extraction_status = "pending"
SORT file.folder, file.name
```

## 추출 완료 (Complete)

```dataview
TABLE WITHOUT ID
  file.link AS "Source",
  file.folder AS "Category"
FROM "03. sources"
WHERE extraction_status = "complete"
SORT file.folder, file.name
```

## 통계

```dataview
TABLE WITHOUT ID
  length(rows) AS "개수",
  key AS "상태"
FROM "03. sources"
WHERE extraction_status
GROUP BY extraction_status AS key
```
