# 코드 가독성과 신경과학

## 📋 기본 정보

- **저자**: Evan Moon
- **발표일**: 2026-01-30

## 🎯 핵심 발견

코드를 읽을 때 뇌에서 활성화되는 영역에 대한 신경과학 연구 정리

### 언어 네트워크 vs Multiple Demand Network

| 구분 | 기존 가정 | 실제 연구 결과 |
|------|----------|---------------|
| **코드 읽기** | 자연어 읽기와 유사 | Multiple Demand Network 활성화 |
| **활성 영역** | Language Network | MDN (논리/추론 영역) |

## 🧠 Multiple Demand Network란?

**Multiple Demand Network(MDN)**가 담당하는 인지 작업:

- 논리적 추론
- 복잡한 인지 작업 처리
- 수학, 퍼즐, 문제 해결
- 공간적 추론

## 💡 시사점

| 관점 | 기존 가정 | 연구 결과 |
|------|----------|----------|
| 코드의 본질 | 언어 | 논리 구조 |
| 가독성 기준 | 잘 쓴 글 | 잘 정리된 논리 |
| 읽기 방식 | 문법/어휘 중심 | 패턴 인식 중심 |

## 🔑 결론

> **코드 가독성**은 "잘 쓴 산문"보다 "잘 정리된 수학 증명"에 가깝다.

### 가독성 향상을 위한 포인트

1. 변수명, 함수명의 언어적 명확성도 중요하지만
2. **논리적 흐름과 구조적 패턴**이 가독성의 핵심
