# 1. 주요 AI 영상 생성 도구 상세 비교

## 1-1. 영상 생성 특화 도구 (Generative Video)

### OpenAI Sora 2
- **출시**: 2024.12 (Sora 1), 2025 (Sora 2)
- **특징**: 텍스트/이미지에서 최대 25초 영상 생성, 동기화된 대화/효과음/음악 동시 생성, 영화적 리얼리즘 최고 수준
- **품질**: 480p~1080p, 물리 시뮬레이션 우수, 프롬프트 충실도 높음
- **가격**:
  - ChatGPT Plus ($20/월): 무제한 480p, 월 50회 생성
  - ChatGPT Pro ($200/월): 10,000 크레딧, 1080p까지
  - 크레딧 팩: $4에 10회 추가 생성
  - API: $0.10/초(720p), $0.30~0.50/초(Pro/1024p)
- **한국 접근**: 가능 (2025.12.31부터 한국 직접 접근 개방)
- **강점**: 시네마틱 품질, 오디오 동기화, OpenAI 생태계 통합
- **약점**: 결과물 일관성 이슈, Pro 아닌 플랜에서 품질 제한, 가격 높음

### Runway Gen-4.5
- **출시**: Gen-4 (2025), Gen-4.5 (2026)
- **특징**: 벤치마크 1위 성능, Motion Brush(움직임 정밀 제어), 장면 일관성 최고, ProRes/PNG 시퀀스 내보내기
- **품질**: 아티팩트 30~40% 적음(경쟁 대비), 프레임 간 흐름 최고, 조명/피사체 연속성 강점
- **가격**:
  - Free: $0 (125 크레딧, Gen-4 Turbo I2V)
  - Standard: $12/월 (625 크레딧, Gen-4.5 포함, 100GB)
  - Pro: $28/월 (2,250 크레딧, 커스텀 음성, 500GB)
  - Unlimited: $76/월 (무제한 생성, Explore 모드)
  - 크레딧 소모: Gen-4 10~12크레딧/초, Turbo 5크레딧/초 (10초 클립 = 120크레딧)
  - 크레딧 이월 불가
- **한국 접근**: 가능 (글로벌 서비스)
- **강점**: 상업용 품질 최고, Motion Brush, 일관성, 에이전시 워크플로우에 최적
- **약점**: 크레딧 소모 빠름, 가격 대비 생성량 적음

### Kling AI 2.6 / 3.0
- **출시**: Kling 2.6 (2025.12), Kling 3.0 (2026.01.31)
- **개발**: Kuaishou (중국)
- **특징**: 영상 최대 3분(체인 연장), 오디오-비주얼 동시 생성(보이스/SFX/앰비언스/랩까지), 1080p 48FPS
- **품질**: 단일 생성 5~10초, 연장으로 3분까지. 물리 액션/캐릭터 미세표정 우수
- **가격**:
  - Free: 매일 66크레딧 (1~6개 짧은 영상/일, 720p)
  - Standard Base(5초/720p): 40크레딧
  - High Quality(1080p): 80크레딧
  - Extended(10초): 80크레딧
  - Pro Max(10초+1080p): 160크레딧
  - 네이티브 오디오(2.6): 기본 대비 5배 크레딧
  - Pro: ~$37/월 (3,000크레딧, HD)
  - 월 구독: $6.99~$180
- **한국 접근**: 가능 (글로벌 서비스, 한국어 UI 부분 지원)
- **강점**: 가성비 최고, 긴 영상, 무료 티어 넉넉, 오디오 통합
- **약점**: 중국 서비스(데이터 이슈 우려), 연장 시 품질 저하 가능

### Pika 2.5
- **출시**: 2025~2026
- **특징**: Pikaswaps(요소 교체)/Pikaffects(효과), 42초 렌더링, 소리 효과 자동 생성(차 충돌 시 금속 소리), 플리커 현상 거의 제거
- **품질**: 캐릭터 정체성/환경 조명 일관성 전문가 수준, 카메라 제어 강화
- **가격**:
  - Free: 제한된 월 크레딧
  - Standard: $8/월
  - Pro: $28/월
  - Fancy: $76/월
- **한국 접근**: 가능 (글로벌 서비스)
- **강점**: SNS 숏폼 최적화(TikTok/Reels/Shorts), 빠른 반복 생성, 바이럴 이펙트, 저렴
- **약점**: 긴 영상 미지원, 시네마틱 품질은 Sora/Runway 대비 부족

### Google Veo 3 / 3.1
- **출시**: Veo 3 (2025), Veo 3.1 (2025.10)
- **특징**: 복합 프롬프트 충실도 벤치마크 1위(MovieGenBench 1,003 프롬프트 테스트), 네이티브 오디오(자연 대화/효과음), 세로형(9:16) 지원, 1080p HD
- **품질**: 시네마틱 스타일, 내러티브 제어, 캐릭터 일관성 우수
- **가격**:
  - Google AI Pro: $19.99/월 (1,000크레딧, 10초 영상 약 125크레딧)
  - Google AI Ultra: $249.99/월 (Veo 3 + Flow 편집기 + Gemini 2.5 Pro + YouTube Premium + 30TB)
  - API: Fast $0.15/초, Standard $0.40/초
- **한국 접근**: 159개국 지원이나 한국 명시적 언급 미확인. Google 서비스 특성상 접근 가능할 가능성 높음
- **강점**: 프롬프트 이해도 최고, Google 생태계 통합, HDR 지원
- **약점**: 가격 높음(Ultra), 독립 플랫폼 아닌 Google 종속

### Luma Dream Machine (Ray3)
- **출시**: Ray3 (2025~2026), Ray3.14 (최신)
- **특징**: 세계 최초 스튜디오급 HDR 네이티브 생성, 4K HDR, 16bit EXR 내보내기, Draft 모드(5배 빠르고 저렴), Modify Video(의상/환경 교체/릴라이팅)
- **품질**: 기본 1080p/24fps, 4K 업스케일 가능, 물리 시뮬레이션 우수
- **가격**:
  - Plus: $23.99/월 (10,000크레딧, 상업 이용 가능)
  - Unlimited: $75.99/월 (10,000크레딧 + Relaxed 무제한)
  - Enterprise: 커스텀
- **한국 접근**: 가능 (글로벌 서비스)
- **강점**: HDR/4K 최고 품질, 프로 워크플로우(EXR), 물리 시뮬레이션
- **약점**: 무료 티어 제한적, 가격 높음

### Hailuo AI (MiniMax)
- **출시**: Hailuo 2.3 (2025~2026)
- **개발**: MiniMax (중국)
- **특징**: 원클릭 영상 생성(Media Agent가 자동 모델 매칭), AI 아바타/음성 내장, 로고/색상/폰트 브랜딩 지원
- **품질**: Hailuo 2.3 Fast 모델로 배치 생성 비용 50% 절감, 물리 액션/스타일화/미세표정 향상
- **가격**:
  - Standard: $9.99/월 (1,000크레딧, 워터마크 제거)
  - Unlimited: $94.99/월 (무제한)
  - 개별 크레딧 구매 가능
- **한국 접근**: 가능 (글로벌 서비스)
- **강점**: 브랜딩 기능 내장, 원클릭 자동화, 가성비
- **약점**: 중국 서비스, Runway/Sora 대비 품질 차이

### PixVerse V5.6
- **출시**: V5.6 (2026)
- **특징**: 네이티브 4K 생성(업스케일 아님), 멀티 스타일(동일 프롬프트로 초현실/애니/스케치), Character-to-Video
- **가격**:
  - Basic: 무료 (90 초기크레딧 + 60 일일크레딧)
  - Standard: $10/월 (1,200크레딧, 720p)
  - Pro: $30/월 (6,000크레딧, 1080p)
  - Premium: $60/월 (15,000크레딧)
  - Enterprise: $100~/월 (API)
- **한국 접근**: 가능
- **강점**: 스타일 다양성, 4K 네이티브, 무료 티어 넉넉
- **약점**: 인지도 낮음, 커뮤니티 소규모

# 2. 2025-2026 최신 트렌드

## 2-1. 기술 발전 방향

### 오디오-비주얼 동시 생성
- Sora 2, Kling 2.6, Veo 3가 선도. 영상+대화+효과음+배경음악을 한 번에 생성
- 후반 작업(음향 추가) 과정이 사라지는 추세

### 해상도/길이 확대
- 4K 네이티브 생성 일반화 (Luma Ray3, PixVerse V5.6)
- 영상 길이: 5초 -> 10초 -> 25초 -> 3분(Kling 체인)으로 확대
- 20초 이상 장편 샷으로 시네마틱 스토리텔링 가능

### 시네마틱 제어
- 카메라 무브먼트(돌리/크레인/핸드헬드/줌)를 촬영 용어로 지시
- "무엇을 만들 수 있나"에서 "얼마나 효과적으로 연출하나"로 경쟁 축 이동

### 실시간 인터랙티브 생성
- 2026 후반 예상: 렌더 대기 없이 실시간으로 장면 조작(가상 카메라, 조명, 캐릭터 표정)
- NVIDIA RTX + LTX-2 + ComfyUI로 PC에서 4K AI 영상 로컬 생성 가능

### 멀티모달 통합
- 텍스트+이미지+오디오+영상 레퍼런스를 한 번에 입력하는 방향
- 단일 프롬프트로 복합 매체 생성

## 2-2. 시장/비즈니스 트렌드

### AI 영상 광고 주류화
- IAB 보고서: 광고 바이어 86%가 AI 영상 제작 사용/계획 중
- 소셜미디어 광고 85%, 디스플레이 73%, TV 56%, 오디오 42%에서 활용
- 2026년까지 소형 브랜드 디지털 영상의 45%가 AI로 제작 전망

### 비용 혁신
- AI 영상 활용 기업: 기존 대비 시간/비용 80% 절감
- LG유플러스: 캐럿 AI 도입 후 광고 제작비 95% 절감
- 제이키즈(아동복): AI 동영상 전환 후 ROAS 2,570% 달성

### AI 전용 SNS 등장
- OpenAI, Meta가 AI 영상 전용 SNS 플랫폼 출시
- AI 크리에이터의 영향력 확대

### 개인 맞춤형 영상 광고
- 타겟에 따라 다른 비주얼/음성 멘트를 자동 노출하는 개인화 영상 광고 전망

### 품질 > 양
- 2026년은 AI 영상의 수는 폭증하되, 기획력/스토리텔링 있는 콘텐츠만 살아남는 시기
- 단순 AI 생성 영상은 관심 못 받음
