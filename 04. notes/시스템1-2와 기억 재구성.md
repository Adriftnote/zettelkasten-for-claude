# 시스템1-2와 기억 재구성

## 맥락

**왜 찾았나**: 카너먼의 시스템1/2가 "빠른 직관 vs 논리적 생각"이 맞는지 확인하려고


## 사실

### 1. 시스템1 (빠른 사고)
자동적, 본능적, 무의식적, 노력 불필요
- 예시: 1+1, 표정으로 감정 파악, 빈 도로 운전
- 출처: [2가지의 사고 모드](https://medium.com/@uxdaysseoul/2가지의-사고-모드-시스템-1-시스템-2-b22a752163ef)

### 2. 시스템2 (느린 사고)
의식적, 분석적, 노력 필요, 천천히 신중하게
- 예시: 437×124 계산, 복잡한 상황 운전, 원인 분석
- 출처: [생각에 관한 생각_대니얼 카너먼](https://3.cpt1st.com/entry/생각에-관한-생각대니얼-카너먼-System-1-System-2)

### 3. 기억은 재생이 아니라 재구성
Elizabeth Loftus 연구: 기억은 비디오 녹화가 아니라 위키피디아처럼 수정 가능
- "smashed" vs "contacted" 실험: 단어 하나가 기억을 바꿈
- 쇼핑몰 미아 실험: 25%가 가짜 기억을 "기억"함
- 출처: [Elizabeth Loftus TED Talk](https://www.ted.com/talks/elizabeth_loftus_the_fiction_of_memory/transcript?language=ko)

### 4. 기억 회상의 실제 과정
1. 시스템1: 감정적으로 강한 조각 빠르게 인출
2. 연상: 유사한 것들 자동 연결 (정확성 검증 없이)
3. 시스템2: 그럴듯한 이야기 구성 (정당화)
- 출처: 대화 중 도출

- 각자 시스템1이 다른 조각을 먼저 인출
- 각자 시스템2가 다른 이야기를 구성
- 둘 다 "검증"이 아니라 "정당화"를 함
