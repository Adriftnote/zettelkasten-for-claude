# 시스템1-2와 기억 재구성

빠른 직관(시스템1)과 느린 분석(시스템2)이 기억 재구성과 결합하여 "확신하지만 틀린 기억"을 만드는 메커니즘.

## 도출 근거

다음 facts의 조합에서 도출됨:

1. **시스템1은 자동적이고 빠르다** - 감정적 조각을 검증 없이 인출
2. **시스템2는 논리적이지만 정당화에 사용된다** - 분석보다 이야기 구성
3. **기억은 재생이 아니라 재구성이다** - Loftus 연구: 위키피디아처럼 수정 가능

→ 따라서: **"빠르게 떠오르는 것" ≠ "정확한 것"**

## Observations

- [fact] 시스템1은 자동적, 본능적, 무의식적으로 작동한다 #cognition
- [fact] 시스템2는 의식적, 분석적이며 노력이 필요하다 #cognition
- [fact] 기억은 비디오 녹화가 아니라 재구성된다 (Loftus) #memory
- [fact] 기억 회상 시 시스템2는 검증이 아닌 정당화를 수행한다 #memory
- [example] 1+1 계산, 표정 읽기는 시스템1 #system1
- [example] 437×124 계산, 복잡한 분석은 시스템2 #system2
- [example] "smashed" vs "contacted" 실험 - 단어 하나가 기억을 바꿈 #loftus
- [example] 쇼핑몰 미아 실험 - 25%가 가짜 기억을 "기억"함 #loftus
- [method] 기억 회상 과정: 시스템1 인출 → 연상 연결 → 시스템2 정당화 #process
- [reference] Elizabeth Loftus TED Talk - The Fiction of Memory #source
- [reference] 대니얼 카너먼 "생각에 관한 생각" #source
