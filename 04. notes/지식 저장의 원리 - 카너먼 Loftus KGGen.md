## 맥락

## 사실

### 1. 잘못된 저장 방식의 문제
감정(시스템1)과 사실이 섞이고, 출처 없이 저장하면:
- 3개월 후: "이거 어디서 봤더라?"
- 6개월 후: 시스템1이 재구성한 "가짜 기억"으로 대체됨
- 출처: Loftus의 구성적 기억 연구

### 2. KGGen의 지식 처리 방식
- 명사(노드): 벡터 임베딩으로 유사성 파악 → 통합
- 동사(관계): 규칙 기반 정규화 → 명시적 연결
- 검증: LLM으로 "같은 것인가?" 확인
- 출처: [[KGGen 이해 - 명사 통합과 동사 관계]]

### 3. 인간 기억 vs KGGen
| | 인간 기억 | KGGen |
|---|----------|-------|
| 인덱싱 | 감정 강도 | 벡터 유사도 |
| 연결 | 자동 연상 | 명시적 관계 |
| 검증 | 없음 (확증편향) | LLM 검증 |
- 출처: 대화 중 도출
