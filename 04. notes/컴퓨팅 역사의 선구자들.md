# 컴퓨팅 역사의 선구자들

프로그래밍을 만든 위대한 인물들의 이야기입니다.

---

## 4.1 찰스 배비지 (Charles Babbage, 1791-1871)

**별명**: "컴퓨터의 아버지"

**주요 업적**:
- **차분 기관 (Difference Engine, 1822)**: 수학 계산표를 자동으로 만드는 기계
- **해석 기관 (Analytical Engine, 1837)**: 프로그래밍 가능한 최초의 범용 컴퓨터 설계

**혁신적인 아이디어**:
```
해석 기관의 구조 (현대 컴퓨터와 동일한 개념!)
┌──────────────────────────────────┐
│ Store (저장소)     → 메모리       │
│ Mill (공장)        → CPU          │
│ Input (펀치카드)   → 입력장치     │
│ Output (프린터)    → 출력장치     │
└──────────────────────────────────┘
```

**재미있는 사실**:
- 증기 기관으로 작동하는 컴퓨터를 구상
- 당시 기술로는 완성하지 못했지만, 설계는 완벽했음
- 2002년 런던 과학박물관이 설계도대로 제작했더니 정상 작동!

## 4.2 에이다 러브레이스 (Ada Lovelace, 1815-1852)

**별명**: "최초의 프로그래머"

**주요 업적**:
- 배비지의 해석 기관을 위한 **최초의 알고리즘** 작성
- 컴퓨터가 단순 계산을 넘어 음악, 그림 등을 만들 수 있을 것이라 예측

**작성한 프로그램**:
베르누이 수를 계산하는 알고리즘 (1843년)
```
Step 1: 변수 초기화
Step 2: 반복문 시작
Step 3: 계산 수행
Step 4: 결과 저장
Step 5: 조건 확인 후 Step 2로 또는 종료
```

**선견지명**:
> "해석 기관은 숫자뿐만 아니라 소리나 그림 같은 어떤 것도 처리할 수 있다"
> - 1843년, 범용 컴퓨팅 개념 제시

**영향**:
- 프로그래밍 언어 **Ada**는 그녀의 이름에서 유래
- 매년 10월 둘째 주 화요일은 "Ada Lovelace Day" (여성 과학기술인의 날)

## 4.3 앨런 튜링 (Alan Turing, 1912-1954)

**별명**: "인공지능의 아버지", "암호 해독의 영웅"

**주요 업적**:

### 1. 튜링 기계 (Turing Machine, 1936)
모든 계산 가능한 문제를 풀 수 있는 이론적 컴퓨터 모델

**구조**:
```
[무한 테이프] ← → [읽기/쓰기 헤드] ← → [상태 기계]
   0 1 0 1 1 0 ...         ↑              규칙 테이블
```

**의미**: 현대 모든 컴퓨터의 이론적 기초

### 2. 제2차 세계대전 암호 해독
- 독일군의 에니그마(Enigma) 암호 해독
- 전쟁을 2년 단축시켜 수백만 명의 생명을 구함
- 영화 <이미테이션 게임> (2014)의 주인공

### 3. 튜링 테스트 (1950)
기계가 지능을 가졌는지 판별하는 테스트

**방법**:
```
[인간 심사관] ← 대화 → [?]
                      ├─ [인간]
                      └─ [컴퓨터]

심사관이 구분 못하면 → 컴퓨터는 지능을 가진 것
```

**현대적 의미**: ChatGPT, Claude 같은 AI의 이론적 토대

## 4.4 존 폰 노이만 (John von Neumann, 1903-1957)

**별명**: "현대 컴퓨터 구조의 창시자"

**주요 업적**: 폰 노이만 구조 (Von Neumann Architecture, 1945)

**핵심 아이디어**: **프로그램과 데이터를 같은 메모리에 저장**

**폰 노이만 구조**:
```
┌─────────────────────────────────────┐
│         메모리 (Memory)              │
│  ┌──────────┬──────────┐            │
│  │ 프로그램 │  데이터  │            │
│  └──────────┴──────────┘            │
└─────────────┬───────────────────────┘
              ↕ (버스)
┌─────────────┴───────────────────────┐
│     CPU (중앙처리장치)               │
│  ┌──────────┬──────────┐            │
│  │ 제어장치 │ 연산장치 │            │
│  └──────────┴──────────┘            │
└─────────────────────────────────────┘
      ↕                    ↕
  [입력장치]          [출력장치]
  키보드, 마우스      모니터, 프린터
```

**혁명적인 이유**:
- 이전: 프로그램 변경 = 하드웨어 재구성 필요
- 이후: 프로그램을 메모리에 로드만 하면 됨

**현대 영향**:
- 스마트폰, 노트북, 서버 모두 폰 노이만 구조 기반
- 70년이 지난 지금도 사용하는 근본 설계

## 4.5 그레이스 호퍼 (Grace Hopper, 1906-1992)

**별명**: "컴파일러의 어머니", "놀라운 그레이스 제독"

**주요 업적**:

### 1. 최초의 컴파일러 개발 (1952)
- A-0 컴파일러: 사람이 읽기 쉬운 언어를 기계어로 변환
- "컴퓨터가 영어를 이해하게 만들 수 있다"고 주장 (당시 조롱받음)

### 2. COBOL 언어 개발 (1959)
비즈니스용 프로그래밍 언어
```cobol
DISPLAY "Hello, World!".
```
- 영어처럼 읽히는 코드
- 은행, 정부 시스템에서 지금도 사용 중

### 3. "버그(Bug)" 용어의 유래
1947년, Harvard Mark II 컴퓨터에서 나방(moth)이 발견됨
→ "버그를 찾았다(found a bug)"라고 기록
→ 프로그램 오류를 "버그"라고 부르게 됨

**명언**:
> "가장 위험한 말은 '우리는 항상 이렇게 해왔어'이다"

## 4.6 클로드 섀넌 (Claude Shannon, 1916-2001)

**별명**: "정보 이론의 아버지", "디지털 시대의 설계자"

**주요 업적**:

### 1. 정보 이론 (Information Theory, 1948)
정보의 양을 수학적으로 측정하는 방법 정립

**핵심 개념**: 비트(Bit) = Binary Digit
```
모든 정보는 0과 1로 표현 가능
- 텍스트: A = 01000001
- 이미지: 픽셀 색상 = RGB(255, 0, 0)
- 음악: 소리 파형을 숫자로 변환
```

### 2. 디지털 회로 설계 (1937)
- 전기 스위치로 논리 연산 수행
- AND, OR, NOT 게이트 개념

**예시**:
```
AND 게이트 (둘 다 1이면 1)
입력 A ─┐
        ├─→ 출력
입력 B ─┘

0 AND 0 = 0
0 AND 1 = 0
1 AND 1 = 1
```

**영향**: 모든 컴퓨터 칩(CPU, GPU)의 기초

---

## 4.7 팀 버너스-리 (Tim Berners-Lee, 1955-)

**별명**: "월드 와이드 웹의 창시자"

**주요 업적**: WWW 발명 (1989)

**만든 것들**:
1. **HTTP** (HyperText Transfer Protocol) - 웹 통신 규칙
2. **HTML** (HyperText Markup Language) - 웹페이지 언어
3. **URL** (Uniform Resource Locator) - 웹 주소
4. **최초의 웹 브라우저** (WorldWideWeb, 1990)
5. **최초의 웹 서버**

**최초의 웹사이트**:
```
http://info.cern.ch/
(1991년 8월 6일 공개, 지금도 접속 가능!)
```

**놀라운 결정**:
- 웹 기술을 **무료로 공개** (특허 출원 안 함)
- 덕분에 인터넷이 전 세계로 확산

**현재**:
- W3C (World Wide Web Consortium) 설립자
- 웹 표준 제정 및 관리

## 4.8 린스 토르발스 (Linus Torvalds, 1969-)

**별명**: "리눅스의 아버지", "오픈소스의 전도사"

**주요 업적**:

### 1. Linux 운영체제 개발 (1991)
21살 대학생 때 취미로 시작
```
"I'm doing a (free) operating system (just a hobby)"
- 1991년 8월 25일, 최초 발표 메시지
```

**리눅스의 영향**:
- 안드로이드 폰 (리눅스 기반)
- 슈퍼컴퓨터 100% 리눅스 사용
- 웹 서버 대부분 리눅스
- NASA 화성 탐사선도 리눅스

### 2. Git 개발 (2005)
버전 관리 시스템 - 코드 변경 이력 추적
```
프로젝트/
  ├── 2024-01-01: 버전 1.0
  ├── 2024-02-01: 버전 1.1 (버그 수정)
  └── 2024-03-01: 버전 2.0 (새 기능)
```

**Git의 영향**:
- GitHub (전 세계 개발자 협업 플랫폼)
- 현대 소프트웨어 개발의 필수 도구
