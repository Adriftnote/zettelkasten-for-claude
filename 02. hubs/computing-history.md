# Computing History

컴퓨팅의 이론적 기초와 역사적 발전을 다루는 허브. 선구자들의 핵심 발명과 그것이 현대 컴퓨팅에 미친 영향을 정리합니다.

## Observations

- [fact] 현대 컴퓨터는 1930-40년대에 정립된 이론(튜링 머신, 폰 노이만 구조)을 그대로 사용 중 #turing #von-neumann
- [fact] "버그"는 Grace Hopper의 나방 발견에서, "컴파일러"는 그녀의 발명에서 유래 #terminology #history
- [decision] 웹(Tim Berners-Lee), 리눅스(Torvalds), Git 모두 무료 공개 결정이 세계적 확산의 핵심 #open-source
