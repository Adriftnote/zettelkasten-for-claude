# Computing History

컴퓨팅의 이론적 기초와 역사적 발전을 다루는 허브. 선구자들의 핵심 발명과 그것이 현대 컴퓨팅에 미친 영향을 정리합니다.

## Observations

### 핵심 인사이트

- [insight] 현대 컴퓨터는 1930-40년대에 정립된 이론(튜링, 폰 노이만)을 그대로 사용 중
- [insight] "버그", "컴파일러" 같은 용어는 특정 인물의 실험/발명에서 유래
- [insight] 웹, 리눅스, Git 모두 "무료 공개" 결정이 세계적 확산의 핵심

### 학습 경로

- [path] 이론적 기초(튜링, 섀넌) → 하드웨어 구조(폰 노이만) → 소프트웨어 도구(호퍼) → 인터넷(버너스-리) → 오픈소스(토르발스)

### 인덱싱 (루만식)

**1. 이론적 기초**
- [index:1] [[Turing Machine (튜링 머신)|튜링 머신]] - 계산 가능성의 이론적 모델 (Alan Turing, 1936)
- [index:1a] [[Turing Test (튜링 테스트)|튜링 테스트]] - 기계 지능 판별 (1950)
- [index:1b] [[Bit]] - 정보의 최소 단위 (Claude Shannon, 1948)
- [index:1c] [[Boolean Algebra (불 대수)|불 대수]] - 논리 연산의 수학적 기초 (George Boole, 1854)

**2. 하드웨어 구조**
- [index:2] [[Von Neumann Architecture (폰 노이만 구조)|폰 노이만 구조]] - 프로그램 내장 방식 (John von Neumann, 1945)
- [index:2a] [[RAM]] - 폰 노이만 구조의 메모리

**3. 소프트웨어 도구**
- [index:3] [[compiler]] - 고급 언어 → 기계어 변환 (Grace Hopper, 1952)
- [index:3a] COBOL - 비즈니스용 언어 (1959)

**4. 인터넷과 웹**
- [index:4] [[HTML (HyperText Markup Language)|HTML]] - 웹페이지 언어 (Tim Berners-Lee, 1989)
- [index:4a] HTTP - 웹 통신 프로토콜

**5. 오픈소스**
- [index:5] Linux - 오픈소스 운영체제 (Linus Torvalds, 1991)
- [index:5a] Git - 버전 관리 시스템 (2005)
