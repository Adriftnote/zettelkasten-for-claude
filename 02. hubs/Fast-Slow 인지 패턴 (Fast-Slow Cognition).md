# Fast-Slow 인지 패턴 Hub

카너먼의 시스템1/2에서 시작해 AI, 메모리, 컴퓨터 구조까지 관통하는 Fast-Slow 이중 구조 패턴을 조직화합니다.

## Observations

- [fact] Fast-Slow 패턴은 도메인을 초월하는 프랙탈 구조 - 인간 사고, AI 메모리, 하드웨어 캐시 모두 동일 #fractal
- [fact] 시스템1(Fast): 자동적, 본능적, 무의식적, 패턴 매칭 #system1
- [fact] 시스템2(Slow): 의식적, 분석적, 노력 필요, 논리적 추론 #system2
- [fact] 모든 효율적 시스템은 Fast(캐시/직관)와 Slow(저장소/분석)의 조합 #architecture
- [fact] GPU는 시스템1의 패턴 인식을 모방, CPU는 시스템2의 순차 분석을 담당 #hardware
- [fact] 기억 회상: 시스템1 인출 → 연상 연결 → 시스템2 정당화 (검증 아님!) #memory
- [method] Fast 레이어가 실패할 때만 Slow 레이어 호출 = 효율성의 핵심 #efficiency
