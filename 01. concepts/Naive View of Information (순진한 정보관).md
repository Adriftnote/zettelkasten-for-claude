# Naive View of Information (순진한 정보관)

**정보가 많을수록 진실에 가까워지고, 더 나은 결정을 내릴 수 있다**는 순진한 믿음입니다.

## 📖 개요

유발 하라리가 《넥서스》에서 제시한 개념으로, 현대 정보 사회의 근본적인 착각을 지적합니다.

```
순진한 정보관의 공식:
더 많은 정보 → 더 많은 진실 → 더 나은 결정 → 더 나은 세상
```

> [!warning] 핵심 문제
> 이 공식은 **정보의 질, 맥락, 해석 능력**을 완전히 무시합니다.

## 🔍 순진한 정보관의 특징

### 기본 전제들

| 전제 | 현실 |
|------|------|
| 정보 = 진실 | 정보는 왜곡, 조작, 오류를 포함 |
| 더 많은 정보 = 더 나은 판단 | 정보 과부하는 판단력을 마비시킴 |
| 정보 접근 = 지식 획득 | 해석 능력 없이 정보는 무의미 |
| 정보 공유 = 민주주의 강화 | 허위정보도 똑같이 확산됨 |

### 역사적 맥락

```
인쇄술 시대: 책 = 지식의 상징 → "더 많은 책 = 더 현명한 사회"
인터넷 시대: 정보 = 민주주의 → "모두가 정보에 접근하면 세상이 좋아진다"
SNS 시대:   공유 = 연결 → "더 많은 연결 = 더 나은 이해"
```

## ⚠️ 왜 위험한가?

### 1. 정보와 진실의 혼동

정보(information)는 **데이터의 패턴**일 뿐, 그 자체로 진실을 보장하지 않습니다.

```
정보: "A가 B라고 말했다"
진실: A가 실제로 그렇게 말했는가? + 그 말이 사실인가?
```

### 2. 정보 과부하의 역설

> [!quote] 하라리의 통찰
> "정보의 홍수 속에서 우리는 더 많이 알게 된 게 아니라,
> 무엇이 중요한지 판단하는 능력을 잃어가고 있다."

### 3. 알고리즘과 필터 버블

- 추천 알고리즘은 **확인 편향**을 강화
- "더 많은 정보"가 아니라 "더 편향된 정보"에 노출
- 정보의 양 ↑, 관점의 다양성 ↓

## 💡 대안: 성숙한 정보관

| 순진한 정보관 | 성숙한 정보관 |
|--------------|--------------|
| 양(quantity) 중심 | 질(quality) 중심 |
| 수집(collecting) | 선별(curating) |
| 접근(access) | 해석(interpretation) |
| 확산(spreading) | 검증(verifying) |

### 필요한 능력들

1. **정보 리터러시** - 출처 확인, 교차 검증
2. **비판적 사고** - 맥락과 의도 파악
3. **주의력 관리** - 무엇에 주의를 기울일지 선택
4. **불확실성 수용** - "모른다"고 말할 수 있는 용기

## 🎯 실천적 시사점

> [!tip] 정보를 대할 때
> - "이 정보의 **출처**는 무엇인가?"
> - "이 정보를 퍼뜨리는 사람의 **목적**은 무엇인가?"
> - "**반대 의견**은 무엇이라고 하는가?"
> - "내가 이것을 믿고 **싶어하는** 이유는 무엇인가?"
