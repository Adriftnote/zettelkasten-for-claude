# Default Mode Network (DMN)

**휴식 상태**나 **내면 지향적 사고** 시 활성화되는 뇌 영역 네트워크입니다. 반추, 공상, 자기 참조적 사고를 담당합니다.

## 📖 개요

Default Mode Network(기본 모드 네트워크)는 외부 과제에 집중하지 않을 때 활성화되는 뇌 영역입니다. 내측 전전두엽, 후측 대상피질, 측두-두정 접합부 등에 분포합니다. 2001년 Marcus Raichle이 명명했습니다.

## 🎭 비유

**백그라운드 앱 비유**:
```
뇌 = 스마트폰

집중할 때 (MDN 활성):
┌─────────────────────┐
│  [코딩 앱]  전면 실행 │
│                     │
│  DMN: 백그라운드 대기 │
└─────────────────────┘

쉴 때 (DMN 활성):
┌─────────────────────┐
│  [DMN 앱] 전면 실행  │
│  - 과거 회상 중...   │
│  - 미래 계획 중...   │
│  - 자기 분석 중...   │
└─────────────────────┘

ADHD:
┌─────────────────────┐
│  [코딩 앱]  전면 실행 │
│  [DMN 앱]  계속 알림! │ ← 안 꺼짐!
│  "어제 그거 왜 그랬지"│
└─────────────────────┘
```

## ✨ 담당 기능

| 기능 | 설명 | 예시 |
|------|------|------|
| **자기 참조** | 나에 대한 생각 | "나는 어떤 사람인가" |
| **정신적 시간여행** | 과거/미래 상상 | 회상, 계획 |
| **사회적 인지** | 타인 마음 이해 | "그 사람은 왜 그랬을까" |
| **마음 방황** | 무목적 사고 | 멍 때리기, 공상 |

## ⚡ DMN vs MDN

```
     DMN                          MDN
  (휴식 모드)                  (집중 모드)
      │                           │
      ▼                           ▼
  ┌────────┐                 ┌────────┐
  │ 내면   │  ←── 반비례 ──→  │ 외부   │
  │ 지향   │      관계       │ 지향   │
  └────────┘                 └────────┘
      │                           │
      ▼                           ▼
  - 반추                      - 논리 추론
  - 공상                      - 문제 해결
  - 자기 분석                 - 코드 읽기
  - 과거/미래                 - 현재 과제
```

| 구분 | DMN | [[Multiple Demand Network|MDN]] |
|------|-----|-----|
| **활성화** | 휴식, 내면 사고 | 집중, 외부 과제 |
| **관계** | 서로 억제 (반비례) | 서로 억제 (반비례) |
| **과활성화 시** | 반추, 우울, ADHD 증상 | (드묾) |

## 🧠 ADHD와 DMN

### 핵심 문제: DMN 억제 실패

```
정상:
  [과제 시작] → DMN OFF → MDN ON → 집중!

ADHD:
  [과제 시작] → DMN ON(유지) → MDN ON → 간섭!
                    ↑
              "딴 생각 침투"
```

### 증상과의 연결

| 증상 | DMN 관련 메커니즘 |
|------|------------------|
| **주의력 결핍** | 집중 시 DMN 억제 안 됨 |
| **마음 방황** | DMN 과활성화 |
| **과거 반추** | DMN의 자기 참조 기능 과잉 |
| **시간 감각 이상** | 정신적 시간여행 기능 이상 |

## 😔 우울증과 DMN

반추(rumination)의 악순환:

```
부정적 사건
    ↓
DMN 활성화 (자기 참조)
    ↓
"내가 뭘 잘못했지..."
    ↓
부정적 감정 ←──────┐
    ↓              │
더 강한 DMN 활성화  │
    ↓              │
"역시 나는..."  ───┘
    ↓
[반추 루프]
```

**연구 결과**: 우울증 환자에서 DMN 과활성화 및 MDN과의 전환 장애 관찰

## 💡 DMN 조절 방법

| 방법 | 효과 | 원리 |
|------|------|------|
| **명상** | DMN 활동 감소 | 현재 순간에 집중 |
| **운동** | DMN-MDN 전환 개선 | 신경 가소성 증가 |
| **인지행동치료** | 반추 패턴 변경 | 사고 재구조화 |
| **외부 과제 집중** | 일시적 DMN 억제 | MDN 활성화 |

## 🔬 연구 역사

- **2001**: Marcus Raichle가 "Default Mode" 명명
- **발견 계기**: fMRI로 "아무것도 안 할 때"도 특정 영역 활성화 발견
- **의미**: 뇌는 쉴 때도 바쁘다 (자기 유지 작업)
