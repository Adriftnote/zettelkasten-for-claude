# Neural Network Layer Depth

신경망에서 입력이 출력에 도달하기까지 거치는 레이어(층)의 수. 깊을수록 더 복잡한 패턴을 학습할 수 있다

## 📖 개요

Neural Network은 여러 개의 레이어(층)를 쌓아 만든다. 입력 데이터는 첫 번째 레이어부터 마지막 레이어까지 순서대로 통과하며, 각 레이어는 데이터를 조금씩 변환한다. "깊이(depth)"란 이 레이어가 몇 개 쌓여 있는지를 말한다. LLM의 경우 Transformer 블록이 수십~수백 개 쌓인 구조이며, 각 블록이 하나의 레이어에 해당한다.

## 🎭 비유

**회사의 결재 라인**을 생각하면 된다.

```
[사원] → [대리] → [과장] → [부장] → [이사] → [사장]
 L1       L2       L3       L4       L5       L6
 얕은 층                                      깊은 층
```

- **얕은 층 (사원~대리)**: 기본적인 일을 처리한다. "이 서류 양식이 맞는지", "오타가 없는지" 같은 표면적 판단.
- **깊은 층 (부장~사장)**: 복잡한 판단을 내린다. "이 계약이 회사 전략에 맞는지", "리스크는 없는지" 같은 고차원 판단.

**쉬운 결재**(문구류 구매)는 대리 선에서 끝나지만 (= 얕은 레이어에서 수렴),
**어려운 결재**(신규 사업 투자)는 사장까지 올라가야 한다 (= 깊은 레이어까지 수렴 안 함).

이것이 바로 DTR이 측정하는 "settling depth"의 핵심 아이디어다.

## ✨ 특징

- **얕은 레이어**: 문법, 품사, 단어 빈도 같은 **표면적 패턴** 처리
- **중간 레이어**: 구문 구조, 문맥 관계, 엔티티 인식 같은 **의미적 패턴** 처리
- **깊은 레이어**: 논리 추론, 수학 계산, 인과 관계 같은 **고차 추론** 수행
- **레이어 수 = 모델 규모**: GPT-3는 96층, LLaMA-70B는 80층 등

## 💡 LLM에서의 레이어 동작 예시

```
입력: "2 + 3 ="

Layer 1~10 (얕은 층):
  → "이건 수학 표현식이다" 인식
  → "+" 기호가 덧셈임을 파악

Layer 11~30 (중간 층):
  → 2와 3이라는 숫자를 내부 표현으로 인코딩
  → 덧셈 연산을 추상적으로 수행 시작

Layer 31~50 (깊은 층):
  → "5"라는 답에 대한 확신이 형성
  → 최종 출력 분포에 수렴

---

입력: "The cat sat on the ___"

Layer 1~5만에 수렴:
  → "mat" (매우 흔한 패턴, 깊이 생각할 필요 없음)
  → settling depth가 낮음 = 쉬운 예측
```

## 🔬 DTR과의 연결

DTR 논문이 레이어 깊이를 활용하는 방식:

1. 각 토큰을 생성할 때, **중간 레이어의 은닉 상태**를 어휘 공간으로 투영
2. 그 투영 결과가 **최종 레이어의 출력과 얼마나 다른지** 비교 (JSD)
3. 일찍 수렴하면 = 쉬운 토큰, 늦게 수렴하면 = 어려운 토큰 (deep-thinking token)

```
[토큰: "and"]  → Layer 5에서 이미 수렴  → 얕은 사고
[토큰: "42"]   → Layer 45까지 수렴 안 함 → 깊은 사고 (deep-thinking)
```
