# 선형결합 (Linear Combination)

벡터들에 각각 계수(스칼라)를 곱하여 더하는 연산. AI의 임베딩, Attention, 표현 학습의 수학적 기초.

## 📖 개요

선형결합은 주어진 벡터들에 계수를 곱해 합산하는 연산(c₁v₁ + c₂v₂ + ... + cₙvₙ)입니다. 행렬곱 Ax = b를 "열벡터의 선형결합"으로 해석하면 AI의 핵심 연산들이 자연스럽게 이해됩니다. 임베딩은 기저 벡터 선택이고, Attention은 계수(가중치) 조절이며, 모델 출력은 결과 벡터입니다.

## 🎭 비유

칵테일 만들기와 같습니다. 럼(벡터1), 라임즙(벡터2), 설탕(벡터3)이 재료이고, 각 재료의 양(계수)을 조절하여 최종 음료(결과 벡터)를 만듭니다. 재료는 같아도 배합 비율에 따라 완전히 다른 맛이 나옵니다.

## ✨ 특징

- **기본 연산**: 선형대수의 거의 모든 연산이 선형결합으로 환원됨
- **행렬곱의 본질**: Ax는 A의 열벡터들을 x의 성분으로 선형결합한 것
- **닫힘 성질**: 벡터 공간 내의 선형결합은 항상 같은 공간에 속함
- **AI 핵심**: 임베딩 조회, Attention 가중합, FFN 변환 모두 선형결합

## 💡 예시

### 행렬곱의 두 가지 시점

```
연립방정식: ax + by = c
            bx + cy = z

시점 1 — 행(row) 기준: "조건 나열"
  (a b)(x)   (c)
  (b c)(y) = (z)    각 행이 하나의 제약 조건

시점 2 — 열(column) 기준: "재료 조합"
      (a)     (b)   (c)
  x · (b) + y·(c) = (z)    열벡터를 재료로, x·y가 배합 비율
```

시점 2가 AI에서 핵심적인 해석입니다.

### AI에서의 선형결합

**임베딩 조회** — 기저 벡터 선택

```
"고양이" → 임베딩 테이블에서 행 조회
         → [0.12, -0.55, 0.03, ..., 0.77]  (4096차원 벡터)

이것은 사실:
  1 · (고양이 벡터) + 0 · (개 벡터) + 0 · (사과 벡터) + ...
  = 원핫 벡터와 임베딩 행렬의 행렬곱 = 열벡터 선택
```

**Attention** — 계수 동적 조절

```
고정 계수:  1.0·(고양이) + 1.0·(가) + 1.0·(귀엽다)
Attention: 0.7·(고양이) + 0.1·(가) + 0.2·(귀엽다)
           ↑ 문맥에 따라 "지금 고양이가 중요"하면 가중치 상승
```

**Diffusion 디노이징** — 반복 선형결합

```
이미지 = Σ (기저 패턴ᵢ × 계수ᵢ)
매 스텝마다 계수를 조정하여 노이즈에서 의미 있는 이미지로 수렴
```

### 대응 관계 정리

| 선형대수 | AI 연산 |
|----------|---------|
| 열벡터 (재료) | 토큰 임베딩 벡터 |
| 계수 (배합 비율) | Attention 가중치 |
| 결과 벡터 | 문맥이 반영된 출력 |
| 선형결합 공간 (Span) | 모델이 표현 가능한 의미의 범위 |
