# 소스코드 (Source Code)

사람이 읽고 쓸 수 있는 형태로 작성된 프로그램의 원본 텍스트입니다.

## 📖 개요

개발자가 텍스트 에디터나 IDE에서 직접 작성하는 코드입니다. 프로그래밍 언어의 문법을 따르며, 사람이 이해할 수 있도록 작성되어야 합니다. 모든 프로그램의 시작점이며, 컴퓨터가 실행하기 전에 컴파일이나 해석을 거쳐야 합니다.

## 🎭 비유

요리사가 작성한 레시피와 같습니다. 사람이 읽고 이해할 수 있도록 명확하게 작성되어 있으며, 이를 따라 실행하면 원하는 결과를 얻을 수 있습니다.

## ✨ 특징

- **텍스트 기반**: `.py`, `.js`, `.java` 등의 파일 확장자
- **사람이 읽을 수 있음**: 주석, 들여쓰기, 변수명 등으로 의도 표현
- **직접 수정 가능**: 코드 변경 후 다시 실행 가능
- **플랫폼 독립적**: 같은 코드를 다양한 환경에서 실행 가능

## 💡 예시

```python
# Python 소스코드
name = "홍길동"
age = 25
print(f"안녕하세요, {name}님입니다. 나이는 {age}살입니다.")
```
