# 기저 변환 (Change of Basis)

동일한 데이터를 다른 좌표계(기저)로 표현하여 숨겨진 구조를 드러내는 선형대수의 핵심 연산

## 📖 개요

기저 변환은 벡터 공간에서 기저(basis)를 교체하는 연산입니다. 같은 벡터라도 어떤 기저로 표현하느냐에 따라 패턴이 보이기도 하고 안 보이기도 합니다. AI에서 "모델이 학습한다"는 것은 본질적으로 데이터에서 의미 있는 기저를 찾아가는 과정이며, 이 관점이 표현 학습(Representation Learning)의 수학적 기반입니다.

## 🎭 비유

같은 산을 정면에서 보면 평범한 언덕이지만, 측면에서 보면 날카로운 절벽이 보입니다. 기저 변환은 "데이터를 바라보는 각도를 바꾸는 것"입니다. AI 학습은 수십억 개의 데이터를 보면서 "어떤 각도에서 봐야 의미 있는 패턴이 드러나는지"를 찾아가는 과정입니다.

## ✨ 특징

- **좌표 변환**: 벡터 자체는 변하지 않고, 표현하는 좌표계만 변경
- **정보 보존**: 가역 변환이면 원래 기저로 복원 가능 (정보 손실 없음)
- **패턴 노출**: 적절한 기저 선택 시 데이터의 구조가 명확해짐
- **행렬 표현**: 기저 변환은 행렬곱 P⁻¹AP로 표현 가능

## 💡 예시

### 수학적 정의

```
표준 기저:  e₁ = (1, 0),  e₂ = (0, 1)
새 기저:    b₁ = (1, 1),  b₂ = (1, -1)

벡터 v = (3, 1)
  표준 기저: 3·e₁ + 1·e₂ = (3, 1)
  새 기저:   2·b₁ + 1·b₂ = 2·(1,1) + 1·(1,-1) = (3, 1)

같은 벡터, 다른 좌표 → 다른 패턴이 보임
```

### AI에서의 대응

| 선형대수 | AI |
|----------|-----|
| 기저 벡터 | 임베딩 공간의 축 (각 차원) |
| 기저 변환 행렬 | 학습된 가중치 행렬 (Weight Matrix) |
| 좋은 기저 찾기 | 모델 학습 (Training) |
| 각 축이 하나의 의미 | 해석 가능한 특징 (Interpretable Feature) |

### AI 학습 = 의미 있는 기저 찾기

```
랜덤 초기화:                학습 후:
  차원1: 의미 없음            차원1: "동물인가?"
  차원2: 의미 없음    →       차원2: "크기가 큰가?"
  차원3: 의미 없음            차원3: "귀여운가?"
```

- SAE: 학습 완료된 모델을 사후 분석하여 해석 가능한 기저를 재발견
- RAE: 이미지 픽셀 공간을 의미 중심 기저로 변환하여 압축
- PCA: 분산이 최대인 방향을 기저로 선택하는 고전적 기저 변환

## Relations

- enables [[Linear Representation Hypothesis]] (수학적 기반 — 개념이 선형 방향으로 표현된다는 것은 특정 기저에서 축 정렬됨을 의미)
- enables [[SAE (Sparse Autoencoder)]] (SAE는 해석 가능한 기저를 찾는 기저 변환)
- relates_to [[Superposition (중첩)]] (중첩 = 현재 기저에서 의미가 뒤엉킨 상태, 기저 변환으로 해소)
- used_by [[선형결합 (Linear Combination)]] (기저 변환의 결과는 새 기저 벡터들의 선형결합으로 표현)
