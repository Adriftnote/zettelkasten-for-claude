# Wisdom (지혜)

**올바른 결정을 내리는 능력**으로 흔히 정의됩니다. 그러나 "올바름"이 규범적 가치에 의존하기 때문에, 지혜 자체도 규범적으로 달라질 수 있습니다.

## 📖 개요

```
일반적 정의:
지혜 = 올바른 결정을 내리는 능력

문제:
"올바름"은 누가, 어떤 기준으로 정의하는가?
```

> [!important] 핵심 통찰
> 지혜는 **규범 중립적(normative-neutral)** 개념이 아니다.
> 특정 가치 체계 안에서만 "지혜롭다"는 판단이 가능하다.

## 📚 철학적 정의들

지혜에 대한 정의는 철학에서 오랫동안 논쟁되어 왔다.

### 아리스토텔레스의 구분

```
Sophia (이론적 지혜)    → 보편적 진리에 대한 이해
                         "세상이 어떻게 작동하는가?"

Phronesis (실천적 지혜) → 구체적 상황에서 올바른 판단
                         "지금 무엇을 해야 하는가?" ← 여기에 규범이 개입
```

### 다섯 가지 주요 이론 (Stanford Encyclopedia of Philosophy)

| 이론 | 핵심 정의 | 규범 의존 정도 |
|------|----------|---------------|
| **인식적 겸손** | 자신의 무지를 아는 것 (소크라테스) | 낮음 |
| **인식적 정확성** | 세계에 대한 정확한 이해 | 낮음 |
| **지식으로서 지혜** | 깊은 진리를 아는 것 | 낮음 |
| **실천적 지혜** | 올바른 결정을 내리는 능력 | **높음** |
| **합리성** | 이성적 판단 능력 | 중간 |

> [!note] 하라리와 일반적 용법
> 《넥서스》를 포함한 대부분의 맥락에서 "지혜"는 **실천적 지혜(Phronesis)**를 가리킨다.
> 따라서 "지혜 = 올바른 결정 → 규범에 의존"이라는 분석은 이 맥락에서 타당하다.

### 최근 연구 동향 (2024-2025)

- **Sternberg (2024)**: 지혜를 통합적으로 봐야 함 - 인지적 요소(지식, 메타인지) + 비인지적 요소(타인에 대한 관심, 감정 조절) + 맥락 적응성
- **Asian Journal of Philosophy (2025)**: 지혜 = 지적 덕(intellectual virtue)의 집합
- **Grossmann et al.**: 균형 잡힌 관점, 인식적 겸손, 맥락 적응성, 다중 관점을 결합한 "공통 이론"

## 🔍 지혜의 규범 의존성

### 구조

```
지혜 → "올바른 결정" → "올바름"의 기준 → 규범/사상
                                    ↓
                        유교? 불교? 공리주의? 자유주의?
```

### 사상별 "지혜"의 차이

| 사상 | "올바름"의 기준 | 지혜로운 결정의 예 |
|------|----------------|-------------------|
| **유교** | 예(禮), 효(孝), 충(忠) | 부모에게 순종, 위계 존중 |
| **불교** | 고통의 소멸, 무집착 | 욕망을 내려놓는 결정 |
| **공리주의** | 최대 다수의 최대 행복 | 전체 효용을 높이는 결정 |
| **자유주의** | 개인의 자율성 존중 | 타인의 자유를 침해하지 않는 결정 |
| **실존주의** | 진정성(authenticity) | 자기 자신에게 솔직한 결정 |

### 예시: 같은 상황, 다른 "지혜"

> 부모가 원하는 직업 vs 내가 원하는 직업

| 관점 | 지혜로운 선택 |
|------|-------------|
| 유교적 | 부모의 뜻을 따르는 것 |
| 자유주의적 | 자신의 선택을 따르는 것 |
| 공리주의적 | 모두의 행복 총합이 큰 쪽 |

→ **같은 행동이 한 규범에서는 지혜롭고, 다른 규범에서는 어리석다**

## ⚠️ 이것이 왜 중요한가?

### 1. "순진한 정보관"의 대안을 다시 생각하게 됨

[[Naive View of Information (순진한 정보관)|순진한 정보관]]의 대안으로 "지혜가 필요하다"고 하지만:

```
순진한 정보관: "정보만 있으면 된다" → 틀림
성숙한 정보관: "정보 + 지혜가 필요하다"
         ↓
질문: "누구의 지혜? 어떤 규범의 지혜?"
```

> [!question] 남는 문제
> 성숙한 정보관도 결국 **어떤 규범을 따를 것인가**라는 질문을 피할 수 없다.

### 2. "지혜로운 AI"의 불가능성?

- AI에게 "지혜롭게 판단하라"고 할 때
- 어떤 규범을 따르게 할 것인가?
- 보편적 지혜가 없다면, AI의 지혜는 **설계자의 규범**을 반영

### 3. 보편적 지혜는 존재하는가?

| 입장 | 주장 |
|------|------|
| **보편주의** | 문화를 초월한 공통 지혜가 있다 (황금률 등) |
| **상대주의** | 모든 지혜는 특정 맥락에 구속된다 |
| **최소주의** | "고통을 줄인다" 같은 최소한의 공통점만 있다 |

## 💡 정보-지식-지혜 위계의 재검토

흔히 말하는 DIKW 피라미드:

```
    지혜 (Wisdom)     ← "올바른" 적용
      ↑
    지식 (Knowledge)  ← 정보의 구조화
      ↑
    정보 (Information) ← 데이터의 의미
      ↑
    데이터 (Data)      ← 원시 사실
```

> [!warning] 문제점
> 이 위계는 "지혜"가 객관적으로 존재하는 것처럼 전제한다.
> 그러나 지혜의 정점에는 **규범 선택**이라는 문제가 숨어있다.

## 🎯 실천적 시사점

지혜가 규범에 의존한다면:

1. **"지혜로운 결정"을 주장할 때 → 자신의 규범적 전제를 명시하기**
2. **다른 사람의 "어리석음"을 비판할 때 → 규범 차이일 가능성 고려하기**
3. **보편적 지혜를 주장할 때 → 그것이 정말 보편적인지 의심하기**
