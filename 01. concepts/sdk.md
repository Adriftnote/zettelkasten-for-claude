# SDK (Software Development Kit)

특정 플랫폼을 위한 소프트웨어 개발 도구 모음입니다.

## 📖 개요

SDK는 특정 플랫폼(Android, iOS, AWS 등)에서 개발하기 위해 필요한 모든 것을 포함합니다. 컴파일러, 라이브러리, 문서, 예제, 디버깅 도구, 에뮬레이터 등이 한 패키지로 제공됩니다. SDK를 설치하면 바로 개발을 시작할 수 있습니다.

## 🎭 비유

프라모델 제작 키트처럼, 개발에 필요한 모든 것이 한 상자에 들어있습니다.

## ✨ 특징

- **완전함**: 개발에 필요한 모든 도구 포함
- **플랫폼 특화**: 특정 플랫폼을 위해 최적화
- **문서 풍부**: 튜토리얼과 API 문서 제공
- **테스트 환경**: 에뮬레이터/시뮬레이터 포함

## 💡 예시

**포함 요소**:
- 컴파일러/인터프리터
- 라이브러리와 프레임워크
- 문서 및 예제 코드
- 디버깅 도구
- 에뮬레이터/시뮬레이터

**예시**:
- **Android SDK**: 안드로이드 앱 개발
- **iOS SDK**: iPhone 앱 개발
- **AWS SDK**: Amazon Web Services 이용
