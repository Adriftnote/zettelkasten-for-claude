# RAM (Random Access Memory)

컴퓨터의 **단기 작업 기억**입니다. CPU가 당장 사용할 데이터를 임시로 저장합니다.

## 📖 개요

RAM은 CPU가 빠르게 접근할 수 있는 휘발성 메모리입니다. 전원이 꺼지면 데이터가 사라지지만, SSD/HDD보다 훨씬 빠릅니다.

## 🎭 비유

**요리사의 주방**:
```
SSD/HDD = 냉장고 (재료 보관, 꺼내는 데 시간 걸림)
RAM     = 도마 위 (지금 쓸 재료만 올려둠)
CPU     = 요리사 (실제 요리)

도마(RAM)가 넓으면 → 재료 많이 펼쳐두고 빠르게 작업
도마가 좁으면     → 냉장고 왔다갔다 (느림)
```

## ✨ 특징

| 특성 | 설명 |
|------|------|
| **빠름** | SSD보다 ~100배 빠른 접근 속도 |
| **휘발성** | 전원 끄면 데이터 사라짐 |
| **랜덤 접근** | 어느 위치든 동일한 속도로 접근 |
| **용량** | 일반적으로 16~64GB |

## 💡 메모리 계층 구조

```
        속도 ↑     용량 ↓
        ──────────────────
        │  CPU 캐시 (L1/L2/L3)  │  ← 가장 빠름, KB~MB
        ├──────────────────────┤
        │        RAM           │  ← 여기! GB 단위
        ├──────────────────────┤
        │      SSD/HDD         │  ← 느림, TB 단위
        └──────────────────────┘
        속도 ↓     용량 ↑
```

## ⚖️ RAM vs 다른 메모리

| | RAM | SSD | [[VRAM]] |
|---|-----|-----|----------|
| **사용자** | CPU | 저장용 | GPU |
| **속도** | ~50GB/s | ~5GB/s | ~1000GB/s |
| **휘발성** | O | X | O |
| **용량** | 16~128GB | 500GB~4TB | 8~24GB |
| **가격** | 중간 | 저렴 | 비쌈 |

## ⚠️ RAM 부족 시

| 현상 | 설명 |
|------|------|
| **스왑 (Swap)** | SSD를 RAM처럼 사용 → 매우 느림 |
| **프로그램 멈춤** | 메모리 부족으로 응답 없음 |
| **OOM 에러** | Out of Memory, 프로그램 종료 |
