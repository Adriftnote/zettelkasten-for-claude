# VRAM (Video RAM)

**GPU 전용 메모리**입니다. 그래픽카드에 달린 RAM으로, GPU가 작업할 데이터를 저장합니다.

## 📖 개요

VRAM은 GPU가 빠르게 접근할 수 있는 전용 메모리입니다. 일반 RAM과 분리되어 있어 GPU 연산에 최적화되어 있습니다.

## 🎭 비유

```
회사 비유:
- CPU = 관리자 (범용 업무)
- GPU = 공장 노동자들 (대량 병렬 작업)
- RAM = 관리자 책상
- VRAM = 공장 작업대

작업대(VRAM)가 넓어야 많은 재료를 펼쳐놓고 빠르게 작업 가능
```

## ✨ [[RAM]] vs VRAM

| 특성 | [[RAM]] | VRAM |
|------|-----|------|
| **사용 주체** | CPU | GPU |
| **위치** | 메인보드 | 그래픽카드 |
| **용량** | 16~128GB | 8~24GB (비쌈) |
| **대역폭** | ~50GB/s | ~1000GB/s |
| **용도** | 일반 프로그램 | 그래픽, AI, LLM |

## 🤖 LLM에서 VRAM이 중요한 이유

| 항목       | VRAM 사용량            |
| -------- | ------------------- |
| 모델 가중치   | 7B 모델 ≈ 14GB (FP16) |
| KV-Cache | 컨텍스트 길이에 비례         |
| 활성화 값    | 배치 크기에 비례           |

```
예시: Llama 70B 모델
- 가중치만: ~140GB (FP16)
- 4bit 양자화: ~35GB
- + KV-Cache (32K 컨텍스트): +8GB

→ 최소 48GB VRAM 필요 (A6000, A100 등)
```

## ⚠️ VRAM 부족 시

| 현상 | 해결책 |
|------|--------|
| OOM (Out of Memory) 에러 | 모델 양자화 (4bit, 8bit) |
| 느린 추론 | 배치 크기 줄이기 |
| 긴 컨텍스트 불가 | 슬라이딩 윈도우 [[Cache]] |
