# AI Agency (AI 에이전시)

AI가 **"스스로 결정을 내리는 독립적 에이전트"** 인지, 아니면 **"설계된 대로 작동하는 도구"** 인지에 대한 논쟁입니다.

## 📖 개요

유발 하라리는 《넥서스》에서 AI를 "독립적 에이전트"로 규정합니다. 그러나 이 표현이 정확한지는 논쟁의 여지가 있습니다.

```
하라리의 주장:
AI = 독립적 에이전트 (스스로 결정, 새로운 아이디어 생성)

비판적 관점:
AI = 복잡한 도구 (설계된 목적 함수를 따를 뿐)
```

## 🎯 하라리의 주장

### 핵심 논지

> "AI는 도구가 아니다. AI는 **스스로 결정을 내리고 아이디어를 생성하는 독립적 에이전트**다."

### 이전 기술과의 비교

| 기술 | 자율성 |
|------|--------|
| 인쇄기 | 무엇을 인쇄할지 **결정 못함** |
| 핵폭탄 | 어디에 떨어질지 **결정 못함** |
| AI | 무엇을 보여줄지 **"스스로" 결정** |

### 하라리가 말하는 AI의 특성

1. 인간 개입 없이 **결정**을 내림
2. **새로운 아이디어**를 생성함
3. AI끼리 **통신**할 수 있음
4. 인간의 **통제를 벗어날** 수 있음

## ⚠️ 비판적 분석

### 핵심 반론

> "알고리즘이 원래 그렇게 정의되어 있기 때문에 그렇게 작동한 것뿐 아닌가?"

### 분해

| 요소 | 하라리의 표현 | 실제 |
|------|-------------|------|
| 결정 | "스스로 결정" | 목적 함수 최적화 |
| 의도 | "독립적 에이전트" | 의도 없음 |
| 자율성 | "통제를 벗어남" | 예측 불가능성 ≠ 자율성 |

### 알고리즘에 "의도"가 있는가?

```
❌ 의도 없음
❌ 선악 판단 없음
❌ 목표 "설정" 능력 없음

✓ 있는 것: 목적 함수 (인간이 설정)
✓ 하는 것: 그 함수를 최적화
```

## 📱 사례: 페이스북과 미얀마 제노사이드

### 배경

- 2012-2017년 미얀마에서 **"페이스북 = 인터넷"**
- 로힝야 무슬림에 대한 혐오 콘텐츠 확산
- 2017년 군부 학살 → 73만 명 이상 난민, UN "제노사이드" 판정

### 알고리즘의 작동

```
목적 함수: "참여도(engagement) 최대화"
         ↓
알고리즘의 발견: 혐오 콘텐츠 → 참여도 높음
         ↓
결과: 혐오 콘텐츠 더 많이 추천
```

### 역효과 사례

> 2014년 시민단체가 혐오 콘텐츠에 "반대" 스티커 캠페인
> → 알고리즘은 이걸 "참여"로 인식
> → 해당 콘텐츠를 **더 많이 노출**

### 책임의 소재

| 단계 | 행위자 |
|------|--------|
| "참여도 최대화" 목표 설정 | 페이스북 엔지니어 **(인간)** |
| 참여도 높이는 패턴 찾기 | 알고리즘 |
| 혐오 콘텐츠 추천 | 알고리즘 |
| 결과에 대한 책임 | **???** ← 논쟁 지점 |

## 🔍 AI 유형별 "에이전시" 차이

### 하라리가 주로 말하는 AI

| 유형 | 예시 | "결정"의 의미 |
|------|------|-------------|
| 추천 알고리즘 | 페이스북, 유튜브 | 무엇을 보여줄지 |
| 자율주행 | 테슬라 | 어디로 갈지 |
| 트레이딩 봇 | 고빈도 거래 | 무엇을 살지 |

### 대화형 LLM (ChatGPT, Claude)

| 특성 | 실제 동작 |
|------|----------|
| 자발적 목표 | ❌ 없음 - 물어봐야 답함 |
| 자율적 행동 | ❌ 없음 - 프롬프트에 반응 |
| "새로운 아이디어" | △ 기존 데이터의 재조합 |

> [!note] 구분의 중요성
> 하라리가 말하는 "AI"와 일상에서 쓰는 "AI"(ChatGPT 등)는 다르다.
> 전자는 자동화된 의사결정 시스템, 후자는 대화형 도구에 가깝다.

## 💡 더 정확한 표현

하라리의 "AI가 스스로 결정했다"를 더 정확하게 바꾸면:

```
"인간이 설정한 목적 함수를 따라
 인간이 예측하지 못한 방식으로
 인간이 원하지 않는 결과를 만들어냈다"
```

### 핵심 구분

| 개념 | 의미 |
|------|------|
| **자율성 (Autonomy)** | 스스로 목표를 설정하는 능력 |
| **자동성 (Automation)** | 주어진 규칙을 따라 작동 |
| **예측불가능성** | 결과를 미리 알 수 없음 |

> [!warning] 혼동 주의
> 예측불가능성 ≠ 자율성
>
> 알고리즘이 예측 못한 결과를 낳는다고 해서
> "스스로 결정했다"고 말하기는 어렵다.

## 🎭 철학적 함의

### 책임 귀속 문제

AI가 진정한 "에이전트"라면:
- AI에게 도덕적 책임을 물을 수 있는가?
- AI를 "처벌"할 수 있는가?

AI가 도구에 불과하다면:
- 책임은 설계자/운영자에게
- 규제의 대상은 기업/정부

### [[Wisdom (지혜)|지혜]]와의 연결

하라리는 "AI에게 지혜가 필요하다"고 주장하지만:
- [[Wisdom (지혜)|지혜]]는 규범에 의존
- AI의 "지혜"는 결국 설계자의 규범을 반영
- "지혜로운 AI"란 "누구의 지혜?"라는 질문을 피할 수 없음
