# 인터프리터 (Interpreter)

소스코드를 한 줄씩 읽으면서 즉시 실행하는 프로그램입니다.

## 📖 개요

인터프리터는 소스코드를 미리 모두 변환하지 않고, 실행 시점에 한 줄씩 읽으면서 바로 실행합니다. Python, JavaScript, Ruby 등이 인터프리터 언어입니다. 개발 중 즉시 결과 확인이 가능합니다.

## 🎭 비유

동시통역사가 말하는 것을 즉시 번역하는 것과 같습니다. 대화는 계속되고, 통역사는 실시간으로 번역합니다.

## ✨ 특징

- **즉시 실행**: 코드 작성 후 바로 실행 가능
- **수정 용이**: 빠른 디버깅과 수정
- **플랫폼 독립성**: 인터프리터만 있으면 실행
- **느린 속도**: 매번 해석하며 실행

## 💡 예시

```python
# Python 코드 (인터프리터 언어)
python script.py
# 결과 즉시 출력
```
