# 라이브러리 (Library)

특정 작업을 수행하기 위해 미리 만들어진 코드 모음입니다.

## 📖 개요

라이브러리는 개발자가 자주 사용하는 기능을 모아 재사용할 수 있도록 제공합니다. 날짜 처리, 수학 계산, 이미지 처리 등 특정 영역의 기능을 제공합니다. 표준 라이브러리(언어 기본 포함)와 외부 라이브러리(별도 설치)가 있습니다.

## 🎭 비유

공구 세트와 같습니다. 필요할 때 꺼내 쓸 수 있는 도구 모음입니다.

## ✨ 특징

- **기능 제공**: 공통된 작업을 미리 구현
- **재사용성**: 여러 프로젝트에서 사용 가능
- **문서 제공**: 사용법을 설명하는 문서 포함
- **검증됨**: 많은 사용자에 의해 검증된 코드

## 💡 예시

- **NumPy**: 과학 계산 (행렬, 배열)
- **Requests**: HTTP 통신 (웹 데이터 가져오기)
- **Pillow**: 이미지 처리
- **Django**: 웹 개발
