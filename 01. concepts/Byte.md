# Byte

## 정의

**Byte (바이트)**는 [[Bit]] 8개를 묶은 단위로, 컴퓨터에서 **실제로 사용하는 최소 단위**입니다.

```
1 Byte = 8 Bits
```

## 표현 범위

8비트로 표현 가능한 경우의 수:

```
2^8 = 256가지

10진수: 0 ~ 255
16진수: 00 ~ FF
2진수: 00000000 ~ 11111111
```

## 예시

### 문자 'A' 저장

```
'A' = 65 (10진수)
    = 0x41 (16진수)
    = 01000001 (2진수)
    = 1 Byte
```

### 숫자 255 저장

```
255 = 0xFF (16진수)
    = 11111111 (2진수)
    = 1 Byte로 표현 가능한 최댓값
```

## 왜 8비트가 1바이트인가?

### 역사적 배경

초기 컴퓨터는 바이트 크기가 제각각이었습니다:

| 연도 | 시스템 | 바이트 크기 |
|------|--------|------------|
| 1950년대 | 일부 IBM | 6 bits |
| 1960년대 | 일부 초기 시스템 | 7 bits |
| **1964** | **IBM System/360** | **8 bits** ← 표준화 |
| 1960년대 | 일부 PDP | 9 bits |

### 8비트가 승리한 이유

#### 1. IBM의 시장 지배력
```
IBM System/360 (1964) → 8비트 채택
    ↓
IBM 시장 점유율 80%
    ↓
다른 회사들도 따라함
    ↓
8비트 = 산업 표준
```

#### 2. 기술적 장점

| 이유 | 설명 |
|------|------|
| **충분한 표현력** | 2^8 = 256가지 (영문 대소문자 + 숫자 + 기호 + 확장) |
| **2의 제곱** | 8 = 2^3 → 하드웨어 설계 간단 |
| **확장성** | 8 → 16 → 32 → 64 (모두 2의 제곱) |
| **균형** | 너무 작지도(6), 크지도(12) 않음 |

## 단위

| 단위 | 크기 | 계산 |
|------|------|------|
| **Byte (B)** | 8 bits | 기본 단위 |
| **Kilobyte (KB)** | 1,024 Bytes | 2^10 |
| **Megabyte (MB)** | 1,024 KB | 2^20 |
| **Gigabyte (GB)** | 1,024 KB | 2^30 |
| **Terabyte (TB)** | 1,024 GB | 2^40 |

**주의:**
- 1 KB = 1,024 Bytes (1,000 아님!)
- 이유: 2^10 = 1,024 (컴퓨터는 2진수 체계)

## UTF에서의 바이트

[[Unicode]] 문자를 저장할 때 바이트 사용:

| 인코딩 | 바이트 단위 | 예: '가' |
|--------|-----------|---------|
| [[UTF-8]] | 1~4 bytes | 3 bytes (EA B0 80) |
| [[UTF-16]] | 2~4 bytes | 2 bytes (AC 00) |
| [[UTF-32]] | 4 bytes | 4 bytes (00 00 AC 00) |

## 메모리 크기 예시

### 텍스트
```
"Hello" (ASCII/UTF-8)
= H(1) + e(1) + l(1) + l(1) + o(1)
= 5 bytes
```

### 이미지
```
1920×1080 픽셀, 24-bit 컬러
= 1920 × 1080 × 3 bytes
= 6,220,800 bytes
= 약 6 MB
```

### 오디오
```
CD 품질: 44.1kHz, 16-bit, 스테레오
= 44,100 samples/초 × 2 bytes × 2 채널
= 176,400 bytes/초
= 약 10 MB/분
```

## 프로그래밍에서의 바이트

### 데이터 타입

| 언어 | 1 byte | 2 bytes | 4 bytes | 8 bytes |
|------|--------|---------|---------|---------|
| C/C++ | char | short | int | long long |
| Java | byte | short | int | long |
| Python | - | - | int (가변) | - |
| Go | byte | int16 | int32 | int64 |

### 예시: C
```c
char c = 'A';        // 1 byte
short s = 1000;      // 2 bytes
int i = 100000;      // 4 bytes
long long l = 1e15;  // 8 bytes
```

## 16진수 표현

바이트는 보통 16진수 2자리로 표현:

```
1 byte = 8 bits = 2 hex digits

예시:
255 = 0xFF
65  = 0x41
0   = 0x00
```

**이유:** 16진수 1자리 = 4비트 → 2자리 = 8비트

## Bit vs Byte 비교

| 항목 | Bit | Byte |
|------|-----|------|
| **크기** | 1 bit | 8 bits |
| **값** | 0 또는 1 | 0 ~ 255 |
| **용도** | 이론, 속도 측정 | 파일, 메모리 크기 |
| **기호** | b | B |
| **예시** | 100 Mbps | 10 MB |

**함정:**
```
100 Mbps 인터넷 속도
= 100 Megabits per second
= 12.5 MegaBytes per second
≠ 100 MB/s
```

## 핵심 요약

바이트는 "컴퓨터의 최소 작업 단위"입니다. [[Bit]]가 "알파벳"이라면, 바이트는 "단어"입니다. 8비트가 표준이 된 건 기술적 우수함보다는 **IBM의 영향력 + 적당한 실용성** 덕분입니다.
