# Bit

## 정의

**Bit (비트)**는 컴퓨터에서 정보를 표현하는 **가장 작은 단위**입니다.

Binary Digit (2진 숫자)의 줄임말입니다.

## 값

비트는 두 가지 상태만 가질 수 있습니다:

```
0 (꺼짐, 거짓, 없음)
1 (켜짐, 참, 있음)
```

## 물리적 표현

| 매체 | 0 | 1 |
|------|---|---|
| **전기** | 낮은 전압 | 높은 전압 |
| **자기** | 북극 방향 | 남극 방향 |
| **광학** | 빛 없음 | 빛 있음 |
| **트랜지스터** | 차단 | 통전 |

## 비트의 조합

비트가 많아질수록 표현 가능한 경우의 수가 늘어납니다:

| 비트 수 | 조합 수 (2^n) | 예시 |
|---------|--------------|------|
| 1 bit | 2 | 0, 1 |
| 2 bits | 4 | 00, 01, 10, 11 |
| 3 bits | 8 | 000 ~ 111 |
| 4 bits | 16 | 0000 ~ 1111 (16진수 1자리) |
| 8 bits | 256 | 00000000 ~ 11111111 ([[Byte]] 1개) |

## 예시: 숫자 표현

10진수를 2진수(비트)로 변환:

```
10진수 5:
  = 4 + 1
  = 2² + 2⁰
  = 101 (2진수)
  = 3 bits 필요

10진수 255:
  = 128 + 64 + 32 + 16 + 8 + 4 + 2 + 1
  = 11111111 (2진수)
  = 8 bits 필요
```

## Byte와의 관계

컴퓨터는 보통 비트를 **8개씩 묶어서** 처리합니다:

```
1 Byte = 8 Bits
```

→ 자세한 내용은 [[Byte]] 참조

## 단위

| 단위 | 크기 | 설명 |
|------|------|------|
| **bit (b)** | 1 bit | 기본 단위 |
| **Kilobit (Kb)** | 1,000 bits | 인터넷 속도 측정 |
| **Megabit (Mb)** | 1,000 Kb | 네트워크 대역폭 |
| **Gigabit (Gb)** | 1,000 Mb | 고속 인터넷 |

**주의:** 
- bit (b) ≠ Byte (B)
- 1 Byte = 8 bits
- 100 Mbps ≠ 100 MB/s (8배 차이!)

## 실용 예시

### 인터넷 속도
```
100 Mbps (Megabits per second)
= 100,000,000 bits/초
= 12,500,000 bytes/초
= 12.5 MB/초
```

### 이미지 깊이
```
1-bit 이미지: 흑백 (0=검정, 1=흰색)
8-bit 이미지: 그레이스케일 (0~255 단계)
24-bit 이미지: 컬러 (RGB 각 8비트)
```

### CPU 아키텍처
```
32-bit CPU: 한 번에 32비트 처리
64-bit CPU: 한 번에 64비트 처리
→ 더 큰 메모리 주소 지정 가능
```

## 비트 연산

프로그래밍에서 비트를 직접 조작:

| 연산 | 기호 | 설명 |
|------|------|------|
| AND | & | 둘 다 1이면 1 |
| OR | \| | 하나라도 1이면 1 |
| XOR | ^ | 다르면 1 |
| NOT | ~ | 반전 |
| Shift | << >> | 비트 이동 |

### 예시
```
  1010 (10)
& 1100 (12)
------
  1000 (8)
```

## 역사

| 시기 | 내용 |
|------|------|
| **1946** | ENIAC - 10진수 사용 |
| **1948** | Claude Shannon - "bit" 용어 최초 사용 |
| **1950년대** | 2진수 컴퓨터 표준화 |
| **1964** | IBM System/360 - 8비트 = 1바이트 표준화 |

## 왜 2진수인가?

### 1. 물리적으로 구현하기 쉬움
```
스위치: 켜짐/꺼짐
트랜지스터: 통전/차단
→ 2가지 상태가 가장 안정적
```

### 2. 10진수는 왜 안 쓰나?
```
10진수 = 10가지 전압 준비
→ 노이즈에 약함
→ 회로 복잡
→ 비실용적
```

### 3. 3진수는?
```
3진수 = 이론상 효율적
→ 하지만 하드웨어 구현 어려움
→ 표준화 안 됨
```

## 핵심 요약

비트는 컴퓨터의 "알파벳"입니다. 0과 1 두 글자로 모든 정보(숫자, 문자, 이미지, 소리)를 표현합니다. 일상에서는 보통 8비트 묶음([[Byte]])을 기본 단위로 사용합니다.
