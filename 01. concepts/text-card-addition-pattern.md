# Text Card Addition Pattern

## 📖 개요

Metabase 대시보드에 텍스트 카드를 추가하여 설명, 지침, 유의사항, 섹션 제목 등의 정보를 표시하는 패턴입니다. 단순 텍스트부터 마크다운 형식의 풍부한 포맷까지 지원하여 대시보드의 가독성과 이해도를 향상시킵니다.

## 🎭 비유

신문의 기사와 제목처럼, 차트(기사) 주위에 텍스트(제목, 설명)을 배치하여 독자의 이해를 돕는 것과 같습니다. 숫자와 그래프만으로는 부족한 컨텍스트를 텍스트로 제공합니다.

## ✨ 특징

- **Rich Text Support**: 마크다운 형식 지원 (굵게, 기울임, 목록 등)
- **Flexible Positioning**: 대시보드의 어느 위치에든 배치 가능
- **Static Content**: 데이터 쿼리 필요 없음 (빠름)
- **Context Enhancement**: 차트의 의미 설명 가능
- **Easy Maintenance**: 간단한 텍스트 편집으로 수정

## 💡 예시

### 예시 1: 대시보드 텍스트 카드 활용

```markdown
## 일일 판매 대시보드 구성

### 구조:

┌─────────────────────────────────────┐
│ 일일 판매 현황                        │  ← 텍스트 카드 (제목)
├─────────────────────────────────────┤
│ 📊 어제 판매액: $45,230              │  ← 텍스트 카드 (요약)
│ ✓ 목표 달성률: 102%                 │
│ ⚠️ 참고: 할인 이벤트 진행 중        │
├─────────────────────────────────────┤
│ [판매 추이 차트]                     │  ← 데이터 카드
│                                     │
├─────────────────────────────────────┤
│ **상품별 판매**                      │  ← 텍스트 카드 (섹션 제목)
├─────────────────────────────────────┤
│ [상품별 판매 테이블]                 │  ← 데이터 카드
│                                     │
├─────────────────────────────────────┤
│ 📌 주의사항                          │  ← 텍스트 카드 (주석)
│ - 반품 처리 중인 주문 3건 포함       │
│ - 배송 지연 주문 15건                │
└─────────────────────────────────────┘

각 텍스트 카드의 위치:
- 맨 위: 대시보드 제목/목적
- 섹션 사이: 섹션 제목 및 설명
- 차트 아래: 해석 지침 및 주의사항
```

### 예시 2: 마크다운 텍스트 카드 예시

```markdown
# 텍스트 카드에 사용할 수 있는 포맷

## 제목 포맷
```
# 큰 제목 (h1)
## 중간 제목 (h2)
### 작은 제목 (h3)
```

## 강조 포맷
```
**굵은 글씨**
*기울임꼴*
~~취소선~~
```

## 목록 포맷
```
- 항목 1
- 항목 2
- 항목 3

1. 첫번째
2. 두번째
3. 세번째
```

## 링크 및 이미지
```
[링크 텍스트](https://example.com)
![대체 텍스트](image-url)
```

## 코드 포맷
```
`인라인 코드`

\`\`\`
코드 블록
여러 줄
\`\`\`
```

## 표 포맷
```
| 열1 | 열2 | 열3 |
|-----|-----|-----|
| A   | B   | C   |
| D   | E   | F   |
```
```

### 예시 3: 실제 대시보드 텍스트 카드

```markdown
## 고객 분석 대시보드 - 텍스트 카드 샘플

### 카드 1: 대시보드 제목
```
# 고객 세분화 분석
2024년 1월 분석 보고서
마지막 업데이트: 2024-01-31 10:30
```

### 카드 2: 섹션 설명
```
## 고객 가치 분석

아래 차트는 고객의 평생 가치(LTV)를 기준으로 고객을 분류한 결과입니다.
- **VIP**: LTV > $50,000
- **프리미엄**: $10,000 < LTV ≤ $50,000
- **일반**: LTV ≤ $10,000

[자세한 분석 방법](https://wiki.example.com/ltv-analysis)
```

### 카드 3: 주요 인사이트
```
## 📊 주요 발견사항

1. **VIP 고객** 152명이 전체 매출의 68% 차지
   - 지난달 대비 +12% 증가
   - 평균 재구매율 85%

2. **신규 고객** 1,240명 유입
   - 지난달 대비 +34% 증가
   - 전환율: 18%

3. **이탈 고객** 89명
   - 6개월 미구매 기준
   - 재활성화 프로그램 권장
```

### 카드 4: 사용 방법
```
## 💡 이 대시보드 사용 방법

### 필터 사용
- **기간**: 분석 기간 선택 (기본값: 최근 1년)
- **지역**: 특정 지역만 보기 (다중 선택 가능)
- **상품군**: 특정 카테고리만 보기

### 차트 해석
마우스를 차트 위에 올리면 상세 정보가 표시됩니다.
특정 항목을 클릭하면 세부 데이터로 드릴다운됩니다.

### 문제 발생 시
질문이나 오류 발생 시: data-team@company.com
```

### 카드 5: 면책 조항
```
## ⚠️ 주의사항

- 데이터는 매일 오전 06:00 업데이트됩니다.
- 어제 판매 데이터는 오후 12:00 이후 반영됩니다.
- 반올림으로 인한 합계 오차 가능 (±1%)

**중요**: 이 대시보드의 데이터는 분석용이며, 공식 재무 보고서로 사용할 수 없습니다.
공식 통계는 재무팀에 문의하세요.
```
```

### 예시 4: 텍스트 카드 추가 단계별 가이드

```markdown
## 텍스트 카드 추가 방법

### 1단계: 대시보드 편집 모드 진입
- 대시보드 우측 상단 "📝 편집" 버튼 클릭
- 편집 모드 진입 확인

### 2단계: 텍스트 카드 추가
- "카드 추가" 또는 "+" 버튼 클릭
- "텍스트" 선택
- 또는 직접 카드를 드래그할 공간 클릭

### 3단계: 텍스트 입력
텍스트 편집기에 내용 입력:

```
# 섹션 제목

내용을 여기에 입력합니다.

- 항목 1
- 항목 2
```

### 4단계: 포맷 적용
- 굵게: **텍스트**
- 기울임: *텍스트*
- 제목: # 또는 ##
- 목록: - 또는 1.

### 5단계: 카드 크기 조정
- 카드의 우측 하단 코너를 드래그하여 크기 조정
- 또는 자동 크기 조정 옵션 사용

### 6단계: 위치 조정
- 카드를 드래그하여 원하는 위치로 이동
- 다른 카드와의 레이아웃 조정

### 7단계: 저장
- "저장" 버튼 클릭
- 변경사항이 대시보드에 적용됨
```

## 🛠️ 해결/적용 방법

### 1단계: 대시보드 구조 계획
```
레이아웃 스케치:
- 최상단: 대시보드 제목 (텍스트)
- 섹션 1: 섹션 제목 (텍스트) + 데이터 카드들
- 섹션 2: 섹션 제목 (텍스트) + 데이터 카드들
- 하단: 설명 및 주의사항 (텍스트)
```

### 2단계: 각 섹션별 텍스트 준비
- 제목: 간결한 표현
- 설명: 2-3줄의 설명
- 주의사항: 불릿 포인트로

### 3단계: 마크다운 작성
- 포맷 검토: 제목, 목록, 강조 등
- 링크 추가 (필요시)

### 4단계: 대시보드에 추가
- 편집 모드 진입
- 각 위치에 텍스트 카드 추가
- 크기와 위치 조정

### 5단계: 검토 및 저장
- 모든 텍스트가 명확한지 확인
- 레이아웃이 균형잡혀 있는지 확인
- 저장
