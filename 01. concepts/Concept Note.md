# Concept Note

하나의 개념을 구조화된 형식으로 정리하는 노트 유형

## 📖 개요

Concept Note는 특정 개념이나 아이디어를 일관된 구조로 문서화하는 노트 형식입니다. 제목, 개요, 비유, 특징, 예시, 관계를 포함하여 개념의 본질을 빠르게 파악하고 다른 지식과 연결할 수 있게 합니다.

## 🎭 비유

백과사전의 한 항목과 같습니다. 개념의 정의, 설명, 예시가 정해진 양식으로 정리되어 있어 누구나 쉽게 이해할 수 있습니다.

## ✨ 특징

- 일관된 구조 (제목 → 개요 → 비유 → 특징 → 예시 → 관계)
- 한 노트 = 하나의 개념 (원자적)
- Relations 섹션에서만 wikilink 사용
- 본문에서는 #태그로 키워드 표시

## 💡 예시

```markdown
# API

프로그램 간 통신 인터페이스

## 📖 개요
API는 서로 다른 소프트웨어가 상호작용하는 방법입니다.

## Relations
- part_of [[백엔드]] (구성요소)
```
