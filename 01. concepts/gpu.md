# GPU (Graphics Processing Unit)

**병렬 처리 전문 프로세서 - 수천 개의 작업을 동시에**

## 📖 개요

GPU는 그래픽 처리를 위해 설계된 프로세서로, **단순한 연산을 대량으로 동시에 처리**하는 데 특화되어 있습니다. 원래는 게임과 3D 그래픽용이었지만, 현재는 AI, 과학 계산, 암호화폐 채굴 등 다양한 분야에서 필수적인 하드웨어가 되었습니다.

## 🎭 비유

**일반 직원 10,000명 vs 천재 수학자 10명**

- CPU = 천재 수학자 10명 (어려운 문제 빠르게)
- GPU = 일반 직원 10,000명 (단순 작업 동시에)

## ✨ 특징

### 1. 대량 병렬 처리

```
CPU:
[강력한 코어] [강력한 코어] ... (8-16개)
→ 복잡한 작업 순차 처리

GPU:
[▪▪▪▪▪▪▪▪▪▪] (2000-10000개 코어)
[▪▪▪▪▪▪▪▪▪▪]
→ 단순한 작업 동시 처리
```

### 2. 전용 메모리 (VRAM)

```
┌───────────────────┐
│   GPU 카드        │
│                   │
│  ┌────┐  ┌────┐  │
│  │GPU │◄─►│VRAM│  │
│  │칩  │  │12GB│  │
│  └────┘  └────┘  │
└───────────────────┘
      ↕
  (PCIe 버스)
      ↕
┌───────────────────┐
│   CPU + RAM       │
└───────────────────┘
```

### 3. 특화된 용도

```
✅ 잘하는 것:
- 그래픽 렌더링
- 이미지/동영상 처리
- AI 학습
- 행렬 연산

❌ 못하는 것:
- 복잡한 논리
- 순차적 작업
- 다양한 종류의 작업
```

## 💡 CPU vs GPU 비교

| 항목 | CPU | GPU |
|------|-----|-----|
| **코어 수** | 8-16개 | 2000-10000개 |
| **코어 성능** | 강력 | 단순 |
| **작업 방식** | 순차적 | 병렬적 |
| **용도** | 범용 | 특화 |
| **비유** | 뇌(전두엽) | 시각 피질 |

## 🎮 GPU 종류

### 1. 내장 GPU (iGPU)

```
위치: CPU 칩 내부
성능: 낮음
용도: 일반 사무, 웹서핑
가격: 무료 (CPU 포함)

예시:
- Intel UHD Graphics
- AMD Radeon Graphics (Ryzen)
```

### 2. 외장 GPU (dGPU)

```
위치: 별도 그래픽 카드
성능: 높음
용도: 게임, 3D, AI, 채굴
가격: 수십만~수백만원

예시:
- NVIDIA GeForce RTX 4090
- AMD Radeon RX 7900 XTX
```

## 📊 실제 성능 비교

### 이미지 처리 (1920x1080 = 207만 픽셀)

```
CPU (순차):
for pixel in pixels {
    process(pixel);  // 하나씩
}
→ 2초

GPU (병렬):
모든 픽셀 동시 처리!
→ 0.01초
```

→ **200배 차이!**

### 행렬 곱셈 (1000x1000)

| 장치 | 시간 |
|------|------|
| CPU (8코어) | 500ms |
| GPU (5000코어) | 5ms |

→ **100배 차이!**

## 🚀 GPU의 활용 분야

### 1. 게임 & 그래픽

```
실시간 렌더링 (60fps):
- 207만 픽셀 × 60 = 초당 1억 2천만 픽셀
- CPU 혼자: 불가능
- GPU: 여유롭게 처리
```

### 2. AI & 딥러닝

```
행렬 연산 대량:
- 이미지 인식
- 자연어 처리
- 강화 학습

GPU가 없으면:
학습 10시간 → 1000시간 (100배 느림)
```

### 3. 과학 계산

```
시뮬레이션:
- 날씨 예측
- 유전자 분석
- 물리 시뮬레이션
```

### 4. 영상 편집

```
4K 영상 렌더링:
- CPU: 10시간
- GPU: 1시간
```

## 🦀 Rust에서 GPU 사용

### CPU 코드

```rust
let mut result = vec![0.0; 1000000];
for i in 0..1000000 {
    result[i] = expensive_calculation(i);
}
// → 순차 처리 (느림)
```

### GPU 코드 (wgpu)

```rust
use wgpu::*;

// 100만 개를 동시에!
gpu.compute_shader(|i| {
    expensive_calculation(i)
});
// → 병렬 처리 (빠름)
```
