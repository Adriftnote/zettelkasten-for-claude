# 원자적 개념 인덱스

**루만 제텔카스텐 원칙**으로 정리된 원자적(Atomic) 개념 모음입니다. 각 개념이 독립적인 노트로 분리되어, 개념 간 연결을 통해 깊이 있는 학습이 가능합니다.

이 폴더는 **전체 지식 베이스의 기초**로 기능하며, 다양한 주제의 문서에서 참고할 수 있습니다.

---

## 📚 개념 분류

### 0️⃣ 웹 기초 개념 (10개)

**웹 3요소 + 핵심 개념**:
- [[HTML (HyperText Markup Language)|HTML]] - 웹 페이지의 구조와 내용
- [[CSS (Cascading Style Sheets)|CSS]] - 웹 페이지의 스타일과 디자인
- [[JavaScript (JS)|JavaScript]] - 웹 페이지의 동적 기능
- [[DOM (Document Object Model)|DOM]] - HTML을 JS가 조작할 수 있는 객체 트리
- [[iframe (Inline Frame)|iframe]] - 페이지 안에 다른 페이지 삽입

**웹 보안 (2개)**:
- [[Same-Origin Policy (동일 출처 정책)|Same-Origin Policy]] - 다른 출처 리소스 접근 제한
- [[CORS (Cross-Origin Resource Sharing)|CORS]] - Cross-Origin 요청 허용 메커니즘

**브라우저 확장 (3개)**:
- [[Chrome Extension|Chrome Extension]] - 브라우저 기능 확장 프로그램
- [[Content Script|Content Script]] - 웹 페이지에서 DOM 접근
- [[Background Script (Service Worker)|Background Script]] - 외부 API 호출 담당

### 1️⃣ 프로그래밍 기초 개념 (18개)

**코드의 종류 (5개)**:
- [[source-code|소스코드]] - 사람이 쓴 원본 코드
- [[script|스크립트]] - 즉시 실행 가능한 코드
- [[program|프로그램]] - 완성된 실행 파일
- [[binary|바이너리]] - 0과 1로 된 기계 파일
- [[machine-language|기계어]] - CPU 직접 실행 코드

**실행 도구 (6개)**:
- [[compiler|컴파일러]] - 전체 코드를 한 번에 변환
- [[interpreter|인터프리터]] - 코드를 한 줄씩 실행
- [[runtime|런타임]] - 프로그램 실행 환경
- [[assembler|어셈블러]] - 어셈블리어 → 기계어
- [[linker|링커]] - 여러 파일을 하나로 연결
- [[virtual-machine|가상 머신]] - 소프트웨어로 만든 컴퓨터

**코드 구조 (6개)**:
- [[module|모듈]] - 기능별 파일 단위
- [[package|패키지]] - 여러 모듈을 폴더로 구조화
- [[library|라이브러리]] - 재사용 가능한 코드 모음
- [[framework|프레임워크]] - 개발 골격과 규칙
- [[sdk|SDK]] - 개발 도구 완전 세트
- [[api|API]] - 소프트웨어 간 통신 규칙
