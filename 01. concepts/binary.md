# 바이너리 (Binary)

기계어. 0과 1로만 이루어진, 컴퓨터가 직접 실행할 수 있는 파일입니다.

## 📖 개요

바이너리 파일은 기계가 직접 이해할 수 있도록 변환된 코드로, 사람이 읽을 수 없습니다. `.exe`, `.app`, `.apk` 같은 실행 파일들이 바이너리 형태입니다. 텍스트 에디터로 열면 알 수 없는 기호들만 보입니다.

## 🎭 비유

완제품 약과 같습니다. 사람이 직접 분석하기 어렵지만, 컴퓨터는 바로 이해하고 실행할 수 있습니다.

## ✨ 특징

- **기계 친화적**: CPU가 직접 이해하고 실행
- **읽기 불가능**: 사람이 내용 파악 불가
- **수정 불가능**: 직접 편집 불가능
- **빠른 속도**: 즉시 실행 가능

## 💡 예시

- Windows 실행 파일: `program.exe`
- Android 앱: `app.apk`
- Mac 애플리케이션: `app.app`
- 16진수 표현: `48 65 6c 6c 6f` (Hello)
