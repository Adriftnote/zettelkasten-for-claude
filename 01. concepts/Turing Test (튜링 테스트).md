# Turing Test (튜링 테스트)

**"기계가 인간과 구분할 수 없을 정도로 지능적으로 대화할 수 있는지 판별하는 테스트"**

Alan Turing이 1950년 논문 "Computing Machinery and Intelligence"에서 제안. "기계가 생각할 수 있는가?"라는 질문을 실용적으로 재정의.

## 📖 개요

핵심 질문: **"기계가 생각할 수 있는가?" → "기계가 인간처럼 보일 수 있는가?"**

```
[튜링 테스트 설정]

┌─────────────┐
│  인간 심판   │ ← 텍스트로만 대화
└──────┬──────┘
       │
       ↓ (질문)
┌──────┴──────┐
│             │
▼             ▼
[인간]      [컴퓨터]

둘 중 누가 컴퓨터인지 판별
→ 구분 못하면 컴퓨터는 "지능적"
```

## 🎭 비유

### 가면 무도회

```
[튜링의 아이디어]

"진짜 지능"을 정의하기 어려움
→ 대신 "구분 불가능"으로 판단

비유:
무도회에서 가면을 쓴 사람이
진짜 귀족인지 평민인지 어떻게 알까?
→ 대화해보고 구분 못하면 귀족으로 취급

"행동이 구분 불가능하면 실질적으로 같다"
```

## 💡 테스트 방식

### 원래 형태: 모방 게임 (Imitation Game)

```
1단계: 남자와 여자, 심판
- 심판이 텍스트로 질문
- 누가 여자인지 맞추기
- 남자는 여자인 척 속이려 함

2단계: 컴퓨터와 인간
- 심판이 텍스트로 질문
- 누가 컴퓨터인지 맞추기
- 컴퓨터는 인간인 척 속이려 함
```

### 현대 형태

```
[표준 튜링 테스트]

참가자: 심판 1명, 인간 1명, 컴퓨터 1명
시간: 보통 5분
방식: 텍스트 채팅만
판정: 심판이 30% 이상 속으면 통과

[질문 예시]
"좋아하는 음악이 뭐예요?"
"어제 뭐 했어요?"
"142857 × 3은?"  ← 함정 (너무 빨리 답하면 의심)
```

## 📊 역사적 시도들

| 프로그램 | 연도 | 전략 | 결과 |
|---------|------|------|------|
| **ELIZA** | 1966 | 심리상담사 역할, 질문 되묻기 | 일부 사람 속임 |
| **PARRY** | 1972 | 편집증 환자 역할 | 정신과 의사 속임 |
| **Eugene Goostman** | 2014 | 13세 우크라이나 소년 역할 | 33% 통과 주장 |
| **ChatGPT/Claude** | 2022+ | LLM 기반 | 많은 상황에서 통과 |

### Eugene Goostman 논란

```
[2014년 "통과" 주장]

전략: 13세 외국인 소년 설정
→ 어색한 대답을 "나이/언어 문제"로 위장

비판:
- 심판이 비전문가
- 5분은 너무 짧음
- 캐릭터 설정이 너무 유리

결론: "진정한 통과"로 인정 안 됨
```

## 💻 현대 LLM과 튜링 테스트

```
[ChatGPT/Claude 시대]

상황: 많은 일상 대화에서 구분 어려움

새로운 질문들:
- 튜링 테스트가 여전히 유효한가?
- "지능"의 정의를 바꿔야 하나?
- 더 어려운 테스트가 필요한가?

제안된 대안:
- Winograd Schema Challenge (상식 추론)
- 물리적 튜링 테스트 (로봇)
- 장기 대화 테스트
```

## ⚠️ 비판과 한계

### 중국어 방 논증 (Chinese Room)

```
[John Searle의 반론, 1980]

시나리오:
- 영어만 아는 사람이 방 안에 있음
- 중국어 규칙책을 보며 기계적으로 답변
- 밖에서 보면 중국어 유창한 것처럼 보임

결론:
"통과해도 진짜 이해하는 게 아닐 수 있다"
튜링 테스트 ≠ 진정한 지능 증명
```

### 기타 비판

| 비판 | 내용 |
|------|------|
| **너무 쉬움** | 속이기만 하면 됨, 진짜 지능 불필요 |
| **인간 중심** | 왜 인간처럼 보여야 지능인가? |
| **텍스트 한정** | 시각, 청각, 물리적 상호작용 무시 |
| **속임수 보상** | 정직보다 속이기에 유리 |

## 💡 튜링의 예측

```
[1950년 튜링의 예측]

"2000년경에는 평균적인 심판이
5분 대화로 컴퓨터를 구분하지 못할
확률이 30%에 달할 것"

결과: 대략 20년 늦게 실현됨 (2020년대)
```

## 🔧 현대적 의의

```
[튜링 테스트의 레거시]

직접적 가치보다 촉발한 논의가 중요:

1. "지능이란 무엇인가?" 철학적 질문
2. AI 연구의 방향성 제시
3. 인간-기계 상호작용 연구
4. AI 윤리와 의식 문제

"통과 여부"보다 "왜 이 테스트인가?"가 핵심
```
