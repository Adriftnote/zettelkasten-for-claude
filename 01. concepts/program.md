# 프로그램 (Program)

컴퓨터가 수행할 작업들의 명령어 집합으로, 사용자가 직접 실행할 수 있는 완성된 소프트웨어입니다.

## 📖 개요

프로그램은 소스코드가 컴파일되거나 패키징되어 최종 사용자가 실행할 수 있는 형태입니다. MS Word, Chrome 브라우저, 카카오톡 같은 실행 파일들이 모두 프로그램입니다. 응용 프로그램, 시스템 프로그램, 유틸리티 등 다양한 종류가 있습니다.

## 🎭 비유

완성된 제품과 같습니다. 공장에서 모든 부품을 조립한 자동차처럼, 프로그램도 모든 코드와 자원이 준비되어 사용자가 바로 사용할 수 있습니다.

## ✨ 특징

- **독립적 실행**: 별도 도구 없이 바로 실행 가능
- **패키징됨**: 필요한 모든 자원 포함
- **배포 가능**: 다른 사람에게 설치 및 실행 가능
- **다양한 종류**: 응용/시스템/유틸리티 프로그램

## 💡 예시

- **응용 프로그램**: Photoshop, Word, Discord
- **시스템 프로그램**: Windows, macOS, Linux
- **유틸리티**: 압축 프로그램, 백신, 파일 관리자
