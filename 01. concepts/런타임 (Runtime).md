# 런타임 (Runtime)

프로그램이 실행되는 환경 또는 실행되는 시점을 의미합니다.

## 📖 개요

런타임은 두 가지 의미를 가집니다. 첫째, 프로그램을 실행시켜주는 소프트웨어 환경(JRE, Node.js 등)을 말하고, 둘째, 프로그램이 실제로 작동하는 시점을 말합니다. 개발자가 작성한 코드는 런타임에 실행됩니다.

## 🎭 비유

영화를 보려면 영화관(하드웨어) 외에도 영사기(런타임)가 필요한 것과 같습니다.

## ✨ 특징

- **실행 환경**: 소프트웨어가 실행되기 위한 필수 요소
- **언어별 상이**: 각 프로그래밍 언어마다 다름
- **메모리 관리**: 프로그램 실행 중 메모리 할당/해제
- **에러 감지**: 런타임에 에러 발견 (0으로 나누기 등)

## 💡 예시

- **Java Runtime Environment (JRE)**: Java 프로그램 실행
- **Node.js Runtime**: JavaScript 실행
- **.NET Runtime**: C# 프로그램 실행
